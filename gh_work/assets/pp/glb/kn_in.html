Game Launcher ಖಾಸಗಿತನ ಸೂಚನೆ


ಪರಿಣಾಮಕಾರಿ ದಿನಾಂಕ: 24 ಅಗಸ್ಟ್ 2021

Samsung Electronics Co., Ltd. (“Samsung”) ನಲ್ಲಿ, ನಮ್ಮ ಗ್ರಾಹಕರಿಗೆ ಖಾಸಗಿತನವು ಎಷ್ಟು ಮುಖ್ಯ ಎಂದು ನಮಗೆ ತಿಳಿದಿದೆ. ನಿಮ್ಮ ವೈಯಕ್ತಿಕ ಮಾಹಿತಿಯನ್ನು ನಾವು ಹೇಗೆ ಸಂಗ್ರಹಿಸುತ್ತೇವೆ ಮತ್ತು ಬಳಸುತ್ತೇವೆ ಎಂಬುದನ್ನು ನೀವು ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವುದನ್ನು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಲು ನಾವು ಈ ಖಾಸಗಿತನ ಸೂಚನೆಯನ್ನು ರಚಿಸಿದ್ದೇವೆ.

https://account.samsung.com/membership/terms/privacypolicy ನಲ್ಲಿರುವ ನಮ್ಮ ಖಾಸಗಿತನ ನೀತಿಯೂ ಸಹ Game Launcher ನ ನಿಮ್ಮ ಬಳಕೆಗೆ ಅನ್ವಯವಾಗುತ್ತದೆ. ನಿಮ್ಮ ಡೇಟಾವನ್ನು ನಾವು ಹೇಗೆ ಬಳಸುತ್ತೇವೆ ಎಂಬುದರ ಕುರಿತು ಹೆಚ್ಚಿನ ಮಾಹಿತಿಯನ್ನು ನಮ್ಮ ಖಾಸಗಿತನ ನೀತಿಯು ಹೊಂದಿದೆ. ನಮ್ಮನ್ನು ಹೇಗೆ ಸಂಪರ್ಕಿಸಬೇಕು ಎಂಬ ಮಾಹಿತಿಯೂ ಸಹ ಅದರಲ್ಲಿದೆ. ಈ ಖಾಸಗಿತನ ಸೂಚನೆಗೆ ಹೆಚ್ಚುವರಿಯಾಗಿ ದಯವಿಟ್ಟು ನಮ್ಮ ಖಾಸಗಿತನ ನೀತಿಯನ್ನು ಓದಿ. ಆದಾಗ್ಯೂ, Game Launcher ಗಾಗಿ ನಿಮ್ಮ ಮಾಹಿತಿಯನ್ನು ನಾವು ಹೇಗೆ ಬಳಸುತ್ತೇವೆ ಎನ್ನುವುದಕ್ಕೆ ಈ ಗೌಪ್ಯತೆ ಸೂಚನೆಯು ಯಾವಾಗಲೂ ಗೌಪ್ಯತೆ ನೀತಿಯ ಮೇಲೆ ಮೇಲುಗೈ ಸಾಧಿಸುತ್ತದೆ.




ನಾವು ಯಾವ ಮಾಹಿತಿಯನ್ನು ಸಂಗ್ರಹಿಸುತ್ತೇವೆ?

Game Launcher ನ ಮೂಲಕ, Samsung ನಿಮ್ಮ ಬಗ್ಗೆ ಮಾಹಿತಿಯನ್ನು ವಿವಿಧ ರೀತಿಗಳಲ್ಲಿ ಪಡೆದುಕೊಳ್ಳುತ್ತದೆ ಮತ್ತು ನಿರ್ವಹಿಸುತ್ತದೆ.

ನೀವು ನೇರವಾಗಿ ಒದಗಿಸುವ ಮಾಹಿತಿ

•ನೀವು Samsung Account ಗೆ ಲಾಗ್ ಇನ್ ಆದಾಗ, ನಿಮ್ಮ Samsung Account ID, ಅಡ್ಡಹೆಸರು, ಪ್ರೊಫೈಲ್ ಚಿತ್ರ, ವಯಸ್ಸು ಮತ್ತು ದೇಶದ ಹೆಸರುಗಳಂತಹ Samsung Account ಮಾಹಿತಿ; ಹಾಗೂ
•ನೀವು ಗ್ರಾಹಕ ಸೇವೆಯನ್ನು ಸಂಪರ್ಕಿಸಿದಾಗ ಉತ್ತರಗಳನ್ನು ಒದಗಿಸಲು, ನಿಮ್ಮ ಗುರುತು ಮತ್ತು ನಮ್ಮನ್ನು ಸಂಪರ್ಕಿಸಲು ಕಾರಣವಾಗಿರುವ ಉತ್ಪನ್ನದ ಕುರಿತು ಪರಿಶೀಲಿಸಲು ಮಾಹಿತಿಗಳನ್ನು ನಾವು ಕೇಳುತ್ತೇವೆ. ನೀವು ನಮ್ಮ ಗ್ರಾಹಕ ಸೇವಾ ಸಿಬ್ಬಂದಿ ಅಥವಾ ಇತರರೊಂದಿಗೆ ನಡೆಸುವ ಸಂವಹನವನ್ನು ನಮ್ಮ ಸಿಸ್ಟಂಗಳ ಮೂಲಕ ರವಾನಿಸಲಾಗುತ್ತದೆ.


Game Launcher ನ ನಿಮ್ಮ ಬಳಕೆಯ ಕುರಿತು ಮಾಹಿತಿ

ನೀವು ಒದಗಿಸುವ ಮಾಹಿತಿಗೆ ಹೆಚ್ಚುವರಿಯಾಗಿ, ನಿಮ್ಮ ಸಾಧನಗಳಲ್ಲಿನ ಸಾಫ್ಟ್‌ವೇರ್‌ ಮೂಲಕ ಮತ್ತು ಇತರೆ ವಿಧಾನಗಳ ಮೂಲಕ ನಾವು ನಮ್ಮ Game Launcher ನ ನಿಮ್ಮ ಬಳಕೆಯ ಕುರಿತು ಮಾಹಿತಿಯನ್ನು ಸಂಗ್ರಹಿಸುತ್ತೇವೆ. ನಾವು ಇವುಗಳನ್ನು ಸಂಗ್ರಹಿಸುತ್ತೇವೆ:

•ನಿಮ್ಮ Game Launcher ಶಕ್ತಗೊಳಿಸಲಾದ ಸಾಧನಗಳು, ಹ್ಯಾಶ್‌ ಮಾಡಲಾದ IMEI, MCC (ಮೊಬೈಲ್ ದೇಶದ ಸಂಕೇತ), ಮತ್ತು MNC (ಮೊಬೈಲ್ ನೆಟ್‌ವರ್ಕ್ ಸಂಕೇತ) ಗಳಂತಹ ಅನನ್ಯ ಸಾಧನ ಗುರುತಿಸುವಿಕೆಗಳು, ಆಪರೇಟಿಂಗ್ ಸಿಸ್ಟಂ ಆವೃತ್ತಿಗಳು ಹಾಗೂ Game Launcher ನ ಸಂಯೋಜನೆಗಳು ಮತ್ತು Game Launcher ಅನ್ನು ಪ್ರವೇಶಿಸಲು ನೀವು ಬಳಸುವ ಸಾಧನಗಳು –ಇವುಗಳೆಲ್ಲದರ ಮಾಹಿತಿ;
•ನೀವು Game Launcher ಅನ್ನು ಹೇಗೆ, ಯಾವಾಗ, ಮತ್ತು ಎಷ್ಟು ಸಮಯಾವಧಿ ಬಳಸುವಿರಿ ಎನ್ನುವುದರ ಕುರಿತು ಮಾಹಿತಿಯನ್ನೂ ಒಳಗೊಂಡಂತೆ ದೋಷ ನಿರ್ಣಯಾತ್ಮಕ, ತಾಂತ್ರಿಕ, ದೋಷದ ಮತ್ತು ಬಳಕೆಯ ಮಾಹಿತಿ, ಹಾಗೂ ಅವುಗಳೊಂದಿಗೆ, Game Launcher ಮೂಲಕ ಒದಗಿಸಲಾಗುವ ವಿಷಯದೊಂದಿಗೆ ನೀವು ಹೇಗೆ ಸಂವಹನ ನಡೆಸುವಿರಿ ಎಂಬಂತಹ Game Launcher ನ ನಿಮ್ಮ ಬಳಕೆಯ ಕುರಿತು ಇತರ ಮಾಹಿತಿ. ಅನುಸ್ಥಾಪಿಸಲಾಗಿರುವ ಆಪ್‌ಗಳ ಶೀರ್ಷಿಕೆಗಳು, ನೀವು ಆಡುತ್ತಿರುವ ಆಟದ ಪ್ಯಾಕೇಜ್ ಹೆಸರು, ಹಾಗೂ ಅನುಸ್ಥಾಪಿಸಲಾಗಿರುವ ಮತ್ತು ಅಳಿಸಲಾಗಿರುವ ಆಟಗಳ ಮಾಹಿತಿಗಳೂ ಒಳಗೊಂಡಂತೆ ನೀವು Game Launcher ನಲ್ಲಿ ಅಥವಾ ಅದರ ಮೂಲಕ ಆಡುವ ಆಟಗಳು ಹಾಗೂ ನಾವು ನಿಮಗಾಗಿ ಶಿಫಾರಸು ಮಾಡುವ ಆಟಗಳ ಕುರಿತು ಮಾಹಿತಿಯನ್ನೂ ಸಹ ನಾವು ಸಂಗ್ರಹಿಸುತ್ತೇವೆ; ಹಾಗೂ
•ಪಾಲುದಾರರಿಂದ ಮಾಹಿತಿ: ನೀವು ಯಾವುದೇ ನಿರ್ದಿಷ್ಟವಾದ ಆಟವನ್ನು ಆಡುವಾಗ, ನೀವು ತೆರೆದಿರುವ ಅಥವಾ ಆಡಲು ಪ್ರಾರಂಭಿಸಿರುವ ಆಟ (ಆಟದ ಆವೃತ್ತಿಯೂ ಸೇರಿದಂತೆ), ಆ ಆಟವನ್ನು ನೀವು ಎಷ್ಟು ಬಾರಿ ತೆರೆಯುವಿರಿ ಅಥವಾ ಆಡಲು ಪ್ರಾರಂಭಿಸುವಿರಿ, ಹಾಗೂ ಯಾವುದೇ ನಿರ್ದಿಷ್ಟ ಆಟವನ್ನು ನೀವು ಎಷ್ಟು ಸಮಯಾವಧಿ ಆಡುವಿರಿ – ಇವುಗಳೆಲ್ಲದರ ಕುರಿತು ಮಾಹಿತಿಯನ್ನು ನಾವು ಪಡೆಯುತ್ತೇವೆ. ಅವುಗಳೊಂದಿಗೆ, ನೀವು ಒಂದು ಟ್ಯುಟೋರಿಯಲ್, ಅಂಕ ಅಥವಾ ಸಾಧನೆಯನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದಾಗ, ನೀವು ಒಂದು ಹಂತವನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದಾಗ, ಮತ್ತು ನೀವು ಯಾವುದೇ ಹಂತದಲ್ಲಿ ವಿಫಲಗೊಂಡಾಗ, ನೀವು ಆಡುವ ಆಟಗಳಲ್ಲಿನ ಕಸ್ಟಮ್ ಸಂದರ್ಭಗಳನ್ನೂ ಸಹ ನಾವು ಸಂಗ್ರಹಿಸುತ್ತೇವೆ.

ಮಾಹಿತಿ ಸಂಗ್ರಹಣೆಯ ಸಮಯದಲ್ಲಿ ನಿಮಗೆ ನಾವು ವಿವರಿಸುವ ರೀತಿಗಳಲ್ಲಿ ಅಥವಾ ನಿಮ್ಮ ಒಪ್ಪಿಗೆಯೊಂದಿಗೆ ನಾವು ನಿಮ್ಮ ಬಗ್ಗೆ, ನಿಮ್ಮ ಸಾಧನಗಳು ಮತ್ತು ಆಪ್‌ಗಳು, ಹಾಗೂ Game Launcher ನ ನಿಮ್ಮ ಬಳಕೆಯ ಕುರಿತು ಇತರೆ ಮಾಹಿತಿಗಳನ್ನೂ ಸಹ ಸಂಗ್ರಹಿಸಬಹುದು.

ನಾವು Firebase Analytics ಗಳಂತಹ ತೃತೀಯ ಪಕ್ಷಗಳ ವಿಶ್ಲೇಷಣಾ ಸೇವೆಗಳನ್ನು ಬಳಸಬಹುದು. ನಾವು ಪಡೆಯುವ ಮಾಹಿತಿಯನ್ನು ಈ ಪೂರೈಕೆದಾರರಿಗೆ ಮತ್ತು ಇತರ ಸಂಬಂಧಿತ ತೃತೀಯ ಪಕ್ಷಗಳಿಗೆ ಬಹಿರಂಗ ಪಡಿಸಬಹುದು ಅಥವಾ ಅವರು ನೇರವಾಗಿ ಸಂಗ್ರಹಿಸಬಹುದು, ಹಾಗೂ ಅವರು ಆ ಮಾಹಿತಿಯನ್ನು Game Launcher ನ ಬಳಕೆಯನ್ನು ಮೌಲ್ಯಮಾಪನ ಮಾಡುವ ಮತ್ತು Game Launcher ಅನ್ನು ನಿರ್ವಹಿಸುವಂತಹ ಕಾರ್ಯಗಳಿಗಾಗಿ ಸಹಾಯ ಮಾಡಲು ಬಳಸಬಹುದು. Firebase Analytics‌ ಕುರಿತು ಇನ್ನಷ್ಟು ತಿಳಿದುಕೊಳ್ಳಲು, ದಯವಿಟ್ಟು https://www.google.com/policies/privacy/partners/ ಗೆ ಭೇಟಿ ನೀಡಿ.





ನಿಮ್ಮ ಮಾಹಿತಿಯನ್ನು ನಾವು ಹೇಗೆ ಬಳಸುತ್ತೇವೆ?

ಸಂಗ್ರಹಿಸಿದ ಮಾಹಿತಿಯನ್ನು ನಾವು ಈ ಕೆಳಗಿನ ಉದ್ದೇಶಗಳಿಗಾಗಿ ಬಳಸುತ್ತೇವೆ:

•ನೀವು Game Launcher ಅನ್ನು ಬಳಸಲು ಅನುವಾಗುವುದಕ್ಕಾಗಿ ನಿಮ್ಮನ್ನು ಗುರುತಿಸಲು ಮತ್ತು ದೃಢೀಕರಿಸಲು;
•ನಿಮಗೆ Game Launcher ಅನ್ನು ಒದಗಿಸಲು;
•ನಿಮಗಾಗಿ ವಿಷಯವನ್ನು ಒದಗಿಸುವುದು ಮತ್ತು ಪ್ರತಿಕ್ರಿಯೆಗಳನ್ನು ವ್ಯೆಯಕ್ತೀಕರಿಸುವಂತಹ Game Launcher ನೊಂದಿಗಿನ ನಿಮ್ಮ ಅನುಭವವನ್ನು ಹಾಗೂ Game Launcher ನೊಂದಿಗೆ ನೀವು ಕಾರ್ಯನಿರ್ವಹಿಸುವ ರೀತಿಗಳನ್ನು ಸುಧಾರಿಸಲು ಮತ್ತು ಗ್ರಾಹಕೀಕರಿಸಲು (ಉದಾಹರಣೆಗಾಗಿ, ನಿಮ್ಮ ವೈಯಕ್ತಿಕ ಸಾಧನೆಗಳು ಮತ್ತು ಪ್ರಗತಿಯನ್ನು ಉಳಿಸಲು ಹಾಗೂ ನಿಮ್ಮ ಹಿಂದಿನ ಚಟುವಟಿಕೆಗಳ ಆಧಾರದಲ್ಲಿ ವೈಯಕ್ತೀಕರಿಸಿದ ಶಿಫಾರಸುಗಳನ್ನು ಒದಗಿಸಲು);
•ನಿಮ್ಮ Samsung account ಅನ್ನು ಸಂಪರ್ಕಿತ ತೃತೀಯ ಪಕ್ಷದ ಸಾಧನಗಳು, ಆಪ್‌ಗಳು, ವೈಶಿಷ್ಟ್ಯಗಳು ಅಥವಾ ಸೇವೆಗಳೊಂದಿಗೆ ಲಿಂಕ್ ಮಾಡಲು;
•Game Launcher ಮೂಲಕ ನೀವು ಮಾಡುವ ವಿನಂತಿಗಳು, ವಿಚಾರಣೆಗಳು ಮತ್ತು ಸೂಚನೆಗಳಿಗೆ ಪ್ರತಿಕ್ರಿಯಿಸಲು;

•ನಿಮ್ಮ ಸಮ್ಮತಿಯನ್ನು ಪಡೆದು ಅಥವಾ ನಿಮ್ಮ ನಿರ್ದೇಶನದ ಮೇರೆಗೆ ಬೇರೆ ಪಕ್ಷಗಳು: ಈ ಖಾಸಗಿತನ ಸೂಚನೆಯಲ್ಲಿ ವಿವರಿಸಲಾಗಿರುವ ಬಹಿರಂಗ ಪಡಿಸುವಿಕೆಗಳಿಗೆ ಹೆಚ್ಚುವರಿಯಾಗಿ, ನೀವು ಪ್ರತ್ಯೇಕವಾಗಿ ಸಮ್ಮತಿಸಿದಾಗ ಅಥವಾ ಅಂತಹ ಹಂಚಿಕೊಳ್ಳುವಿಕೆಯನ್ನು ವಿನಂತಿಸಿದಾಗ ನಾವು ನಿಮ್ಮ ಕುರಿತಾಗಿ ಮಾಹಿತಿಯನ್ನು ತೃತೀಯ ಪಕ್ಷಗಳೊಂದಿಗೆ ಹಂಚಿಕೊಳ್ಳಬಹುದಾಗಿದೆ.




ಈ ಖಾಸಗಿತನ ಸೂಚನೆಗೆ ನವೀಕರಣಗಳು

Game Launcher ನಲ್ಲಿ ನಿಮ್ಮ ಮಾಹಿತಿಯನ್ನು ನಾವು ಹೇಗೆ ಸಂಗ್ರಹಿಸುತ್ತೇವೆ ಮತ್ತು ಸಂಸ್ಕರಿಸುತ್ತೇವೆ ಎಂಬುದನ್ನು ಅಥವಾ ಸಂಬಂಧಿತ ಕಾನೂನುಗಳಲ್ಲಿನ ಬದಲಾವಣೆಗಳ ಬಗ್ಗೆ ನಿಮಗೆ ತಿಳಿಸಲು ಈ ಖಾಸಗಿತನ ಸೂಚನೆಯನ್ನು ನವೀಕರಣಗೊಳಿಸಲಾಗಬಹುದು. ಈ ದಸ್ತಾವೇಜನ್ನು ಕೊನೆಯದಾಗಿ ನವೀಕರಣಗೊಳಿಸಿದ ದಿನಾಂಕವನ್ನು ಈ ಖಾಸಗಿತನ ಸೂಚನೆಯ ಮೇಲ್ಭಾಗದಲ್ಲಿ ತೋರಿಸಲಾಗುತ್ತದೆ.
