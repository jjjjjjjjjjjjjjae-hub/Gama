ประกาศเกี่ยวกับความเป็นส่วนตัวสำหรับ Game Launcher


วันที่มีผลบังคับใช้: 24 สิงหาคม 2021

ที่ Samsung Electronics Co., Ltd. (“Samsung”) เราทราบดีว่าความเป็นส่วนตัวมีความสำคัญกับลูกค้าของเรา เราได้จัดทำประกาศความเป็นส่วนตัวนี้ เพื่อทำให้แน่ใจว่าคุณเข้าใจวิธีการที่เราเก็บรวบรวมและใช้ข้อมูลส่วนบุคคลของคุณ

สามารถดูนโยบายความเป็นส่วนตัวของเราได้ที่ https://account.samsung.com/membership/terms/privacypolicy ซึ่งจะมีผลบังคับใช้กับการใช้งาน Game Launcher. ด้วย นโยบายความเป็นส่วนตัวของเรามีข้อมูลเพิ่มเติมเกี่ยวกับวิธีที่เราใช้ข้อมูลของคุณ และยังมีข้อมูลเกี่ยวกับวิธีการติดต่อเรา โปรดอ่านนโยบายความเป็นส่วนตัวของเรา นอกเหนือจากประกาศความเป็นส่วนตัวฉบับนี้ อย่างไรก็ตาม ประกาศเกี่ยวกับความเป็นส่วนตัวฉบับนี้ จะมีผลบังคับใช้เหนือกว่านโยบายความเป็นส่วนตัวที่เกี่ยวข้องกับวิธีการที่เราใช้ข้อมูลของคุณสำหรับ Game Launcher




เราเก็บรวบรวมข้อมูลอะไรบ้าง

Samsung จะได้รับและเก็บรักษาข้อมูลเกี่ยวกับคุณในรูปแบบต่าง ๆ ผ่าน Game Launcher

ข้อมูลที่คุณให้ไว้โดยตรง

•เมื่อคุณเข้าสู่ระบบ Samsung account ของคุณ ข้อมูลบน Samsung account ของคุณ เช่น ID ของ Samsung account ชื่อเล่น รูปโปรไฟล์ อายุ และประเทศของคุณ และ
•เพื่อทำการตอบรับเมื่อคุณติดต่อฝ่ายบริการลูกค้าของเรา เราต้องขอทราบข้อมูลเพื่อยืนยันตัวตนของคุณและผลิตภัณฑ์ที่เกี่ยวข้องกับเรื่องที่คุณต้องการติดต่อเรา เมื่อคุณสื่อสารกับเจ้าหน้าที่ฝ่ายบริการลูกค้าหรือบุคคลอื่น ๆ ข้อมูลที่คุณแจ้งจะถูกส่งต่อผ่านระบบของเรา


ข้อมูลเกี่ยวกับการใช้ Game Launcher ของคุณ

นอกเหนือจากข้อมูลที่คุณให้ เราจะเก็บรวบรวมข้อมูลเกี่ยวกับการใช้งาน Game Launcher ของคุณผ่านทางซอฟต์แวร์บนอุปกรณ์ของคุณและด้วยวิธีการอื่น ๆ เราจะเก็บรวบรวม:

•ข้อมูลเกี่ยวกับอุปกรณ์ที่ใช้งาน Game Launcher ของคุณ, ตัวระบุอุปกรณ์เฉพาะของคุณ เช่น IMEI ที่เป็นค่าแฮช, MCC (รหัสประเทศของโทรศัพท์มือถือ) และ MNC (รหัสเครือข่ายมือถือ), เวอร์ชันระบบปฏิบัติการ และการตั้งค่า Game Launcher และอุปกรณ์ที่คุณใช้เพื่อเข้าใช้งาน Game Launcher
•ข้อมูลการวิเคราะห์ ทางเทคนิค ข้อผิดพลาด และการใช้งาน รวมถึงการใช้งาน Game Launcher อย่างไร เมื่อใด และนานแค่ไหน รวมถึงข้อมูลอื่น ๆ เกี่ยวกับการใช้ Game Launcher ของคุณ เช่น วิธีที่คุณโต้ตอบกับคอนเทนท์ที่นำเสนอผ่าน Game Launcher นอกจากนี้ เรายังเก็บรวบรวมข้อมูลเกี่ยวกับเกมที่คุณเล่นหรือทางผ่าน Game Launcher และเกมที่เราแนะนำต่อคุณ เช่น ชื่อแอพที่ติดตั้ง ชื่อแพ็กเกจของเกมที่คุณกำลังเล่น และเกมที่ติดตั้งและลบ และ
•ข้อมูลจากพันธมิตร: เมื่อคุณเล่นเกมใดเกมหนึ่ง เราจะได้รับข้อมูลเกี่ยวกับเกมที่คุณเปิดหรือเริ่มเล่น (รวมถึงเวอร์ชันของเกม) จำนวนครั้งที่คุณเปิดหรือเริ่มเล่น และระยะเวลาที่คุณเล่นเกมนั้น ๆ นอกจากนี้ เรายังเก็บรวบรวมเหตุการณ์ที่กำหนดเองในเกมที่คุณเล่น เช่น เมื่อคุณทดลองเล่น ได้คะแนน หรือประสบความสำเร็จ เมื่อคุณผ่านด่านและเมื่อคุณไม่ผ่านด่าน

นอกจากนี้ เรายังอาจเก็บรวบรวมข้อมูลอื่น ๆ เกี่ยวกับคุณ อุปกรณ์และแอปของคุณ และการใช้งาน Game Launcher ของคุณในลักษณะที่เราอธิบายให้คุณทราบในเวลาที่มีการเก็บรวบรวมหรือในกรณีอื่นใดโดยได้รับความยินยอมจากคุณ

เราอาจใช้บริการวิเคราะห์ของบุคคลที่สาม เช่น Firebase Analytics ข้อมูลที่เราได้รับอาจถูกเปิดเผยหรือรวบรวมโดยตรงจากผู้ให้บริการเหล่านี้และบุคคลที่สามที่เกี่ยวข้องอื่น ๆ ที่ใช้ข้อมูล เช่น เพื่อประเมินการใช้ Game Launcher และเพื่อช่วยในการจัดการ Game Launcher หากต้องการเรียนรู้เพิ่มเติมเกี่ยวกับ Firebase Analytics โปรดเยี่ยมชม https://www.google.com/policies/privacy/partners/





เราใช้ข้อมูลของคุณอย่างไร

เราใช้ข้อมูลที่เราเก็บรวบรวมเพื่อวัตถุประสงค์ต่อไปนี้:

•เพื่อระบุและพิสูจน์ตัวตนของคุณ เพื่อให้คุณสามารถใช้ Game Launcher ได้
•เพื่อให้บริการ Game Launcher กับคุณ
•ปรับปรุงและปรับแต่งประสบการณ์ Game Launcher เฉพาะสำหรับคุณ ตัวอย่างเช่น การจัดส่งคอนเทนท์และการตอบสนองที่เหมาะกับคุณ และวิธีการที่คุณติดต่อกับ Game Launcher (ตัวอย่างเช่น บันทึกความสำเร็จและความคืบหน้าส่วนบุคคลของคุณ และให้คำแนะนำส่วนบุคคลเกี่ยวกับเกมบนมือถือตามกิจกรรมที่ผ่านมาของคุณ)
•เพื่อเชื่อมโยง Samsung account ของคุณเข้ากับอุปกรณ์ แอพ คุณสมบัติ หรือบริการของ Samsung หรือบุคคลที่สาม
•เพื่อตอบสนองต่อคำขอ คำถาม และคำสั่งที่เกิดขึ้นผ่า Game Launcher
•เพื่อแสดงโฆษณา เช่น การแสดงโฆษณาที่ปรับตามต้องการ คอนเทนท์ของสปอนเซอร์ และข่าวสารข้อมูลส่งเสริมการขาย (เช่น ผ่าน Samsung Ads: สามารถดูข้อมูลเพิ่มเติม ซึ่งรวมถึงข้อมูลเพิ่มเติมที่ได้รับ วิธีการใช้งาน และวิธีการเลือกไม่รับ Samsung Ads ได้ที่ https://policy.samsungrs.com/policy_en_sea.html)
•เพื่อดำเนินการ ประเมิน และปรับปรุง Game Launcher และธุรกิจของเรา (รวมถึงการพัฒนาผลิตภัณฑ์และบริการใหม่ ขยายและปรับปรุงผลิตภัณฑ์และบริการของเรา จัดการเรื่องการสื่อสารของเรา และวิเคราะห์ผลิตภัณฑ์ บริการ และฐานลูกค้าของเรา ทำการสำรวจตลาด วิเคราะห์ข้อมูล และการปฏิบัติหน้าที่ทางการบัญชี การตรวจสอบ และหน้าที่ภายในอื่น ๆ)
•เพื่อให้การคุ้มครอง ระบุ และป้องกันการหลอกลวง และการกระทำที่เป็นอาชญากรรมอื่น ๆ การร้องเรียน และความรับผิดอื่น ๆ และ
• เพื่อปฏิบัติตามข้อกำหนดตามกฎหมายที่นำมาใช้บังคับ มาตรฐานอุตสาหกรรมที่เกี่ยวข้อง และนโยบายของเรา รวมถึงประกาศแจ้งความเป็นส่วนตัวนี้และนโยบายความเป็นส่วนตัว

เราใช้และรวมข้อมูลที่เราเก็บรวบรวมมาเกี่ยวกับคุณจาก Game Launcher พร้อมด้วยข้อมูลจากบริการ ฟีเจอร์ หรืออุปกรณ์อื่น ๆ ที่คุณใช้ อย่างเช่น เมื่อคุณใช้แท็บ Galaxy Store/Samsung Free Play คุณสามารถรับบริการที่กำหนดเองได้ ซึ่งขึ้นอยู่กับข้อมูลที่ซึ่งรวมเอาบันทึกการใช้เกมจากทั้งบริการ Game Launcher และ Galaxy Store/Samsung Free

Game Launcher อาจสามารถนำเสนอความสามารถในการเชื่อมต่อกับหรือแสดงเว็บไซต์ แอพ บริการอื่น ๆ หรือฟีเจอร์อื่น ๆ อย่างเช่น การแสดงคอนเทนท์ที่คุณชอบหรือการใช้งานจากบริการของ Samsung หรือบุคคลภายนอก เว็บไซต์ แอพ บริการ และฟีเจอร์ต่าง ๆ ของบุคคลที่สามอาจดำเนินการเป็นอิสระจากเรา และอาจมีประกาศและนโยบายความเป็นส่วนตัวของตัวเอง โปรดอ่านประกาศและนโยบายดังกล่าวเพื่อที่คุณจะเข้าใจถึงวิธีการนำข้อมูลของคุณไปใช้งานของพวกเขา ในกรณีที่มีเว็บไซต์ แอป บริการ หรือฟีเจอร์ซึ่งถูกลิงก์หรือแสดงขึ้น ที่ไม่ได้เป็นของเราหรือควบคุมโดยเรา เราจะไม่รับผิดชอบต่อเนื้อหา การใช้ หรือวิธีปฏิบัติด้านความเป็นส่วนตัวของรายการที่กล่าวนั้น




การเชื่อมโยงบัญชี

บุคคลที่สามบางรายอาจนำเสนอบริการเชื่อมโยงบัญชีของ ซึ่งช่วยให้คุณสามารถเชื่อมต่อ Samsung account ของคุณกับบัญชีบริการของบุคคลที่สาม เมื่อเชื่อมโยงบัญชีของคุณแล้ว คุณสามารถใช้คุณสมบัติบางอย่างจากบริการของบุคคลที่สามได้โดยไม่ต้องติดตั้งแอพพลิเคชั่นของบุคคลที่สามนั้น ตัวอย่างเช่น หากคุณลงเข้าสู่ระบบ Samsung account ของคุณและเสร็จสิ้นขั้นตอนการเชื่อมโยงในบริการที่เกี่ยวข้องแล้ว คุณจะสามารถใช้ส่วนหนึ่งของฟังก์ชั่นบริการของบุคคลที่สามที่นำเสนอโดยบุคคลที่สามได้

แต่ละครั้งที่มีการเชื่อมโยงบัญชี ข้อมูลส่วนบุคคลบางรายการจะถูกส่งไปยังบุคคลที่สาม และบุคคลที่สามนั้นจะส่งข้อมูลส่วนบุคคลบางรายการมายัง Samsung เช่นกัน ก่อนที่จะมีการเชื่อมโยงบัญชี เราจะแจ้งให้คุณทราบโดยเฉพาะที่เกี่ยวกับข้อมูลส่วนบุคคลซึ่งจะถูกแชร์และวัตถุประสงค์ของการแชร์ข้อมูลส่วนบุคคลนั้น

เมื่อคุณเชื่อมโยงบัญชี เราจะแจ้งวิธีการลบบัญชีให้คุณทราบ




เราเปิดเผยข้อมูลของคุณต่อใคร

เราจะเปิดเผยข้อมูลของคุณภายในธุรกิจของเราและต่อหน่วยงานดังต่อไปนี้ แต่เพื่อวัตถุประสงค์ตามที่อธิบายไว้ข้างต้นเท่านั้น

•พันธมิตรทางธุรกิจ: พันธมิตรที่เราทำงานร่วมกันเพื่อให้บริการ Game Launcher กับคุณ เช่น บริการของบุคคลที่สามที่จะให้คุณเชื่อมโยงบัญชีของพวกเขากับ Samsung account พันธมิตรทางธุรกิจเหล่านี้จะเป็นผู้ควบคุมและจัดการข้อมูลส่วนบุคคลของคุณ
•ผู้ให้บริการ: บริษัทที่ได้ผ่านการคัดเลือกที่ให้บริการสำหรับเราหรือในนามของเรา เช่น แพลตฟอร์ม Cloud ที่เก็บรักษาข้อมูลในนามของเรา ผู้ให้บริการเหล่านี้มีความมุ่งมั่นที่จะปกป้องข้อมูลของคุณด้วยเช่นกัน
•ฝ่ายอื่น ๆ ตามที่กฎหมายกำหนดหรือตามความจำเป็นเพื่อปกป้อง Game Launcher: ตัวอย่างเช่น มีความจำเป็นตามที่กฎหมายกำหนด กระบวนการทางกฎหมาย หรือคำสั่งศาลจากหน่วยงานภาครัฐในการเปิดเผยข้อมูลของคุณ หน่วยงานเหล่านี้ยังสามารถขอข้อมูลของคุณจากเราเพื่อจุดประสงค์ในการบังคับใช้กฎหมาย ความมั่นคงของชาติ การต่อต้านการก่อการร้าย หรือปัญหาอื่น ๆ ที่เกี่ยวข้องกับความมั่นคง
•ฝ่ายอื่น ๆ ที่เกี่ยวข้องกับการทำธุรกรรมของบริษัท: เราอาจเปิดเผยข้อมูลของคุณต่อบุคคลที่สามซึ่งเป็นส่วนหนึ่งของการควบรวมกิจการ การโอน การเข้าซื้อหรือการขาย หรือในกรณีที่มีการล้มละลาย และ
•ฝ่ายอื่น ๆ ที่ได้รับความยินยอมจากคุณหรือตามคำสั่งของคุณ: นอกเหนือจากการเปิดเผยข้อมูลตามที่ได้อธิบายไว้ในประกาศความเป็นส่วนตัวนี้ เราอาจเปิดเผยข้อมูลเกี่ยวกับคุณต่อบุคคลที่สามหากคุณให้ความยินยอมแยกต่างหากหรือขอให้มีการเปิดเผขข้อมูลดังกล่าว
