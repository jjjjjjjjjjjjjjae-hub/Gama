Game Launcher кызматынын жекелик эскертүүсү


Качан күчүнө кирет: 24-август, 2021-жыл

Samsung Electronics Co., Ltd. компаниясы (мындан ары “Samsung”) купуялык маселесинин кардарлар үчүн канчалык маанилүү экендигин жакшы түшүнөт. Ушул купуялык эскертүүсү аркылуу жеке маалыматыңыздын кандайча топтолуп колдонула турганын биле аласыз.

https://account.samsung.com/membership/terms/privacypolicy дарегинде жайгашкан Жекелик саясаты Game Launcher колдонууңуз карата да жаратылат. Купуялык саясатында маалыматыңыздын кандайча колдонула тургандыгы кененирээк айтылат. Мындан тышкары, биздин байланыш маалыматыбызды да таба аласыз. Купуялык эскертүүсүнө кошумча, Купуялык саясатын да окуп чыгыңыз. Ошону менен бирге, бул Жекелик эскертүүсү Game Launcher кызматына байланыштуу маалыматыңызды колдонуубузда Жекелик саясатына үстөмдүк кылат.




КАНДАЙ МААЛЫМАТ ТОПТОЛОТ?

Game Launcher кызматы аркылуу Samsung компаниясы сиз тууралуу маалыматына ар кандай жолдор менен ээ болуп жана бир калыпта кармап турат.

Өзүңүз берген маалымат

• Samsung Account’ка киргенде, Samsung account ID, ылакап атыңыз, профилиңиздин сүрөтү, жашыңыз жана жашаган өлкөңүз сыяктуу Samsung Account байланыштуу маалымат; жана
• Кардарларды тейлөө кызматына кайрылганыңызда сизге жооп берүү үчүн бизге кайрылуунун себебиндеги өнүмдү жана өздүгүңүздү ырастоого маалыматты сурайбыз. Кардарларды тейлөө кызматындагы жаамат же башкалар менен болгон кабарлашууларыңыз системаларыбыз аркылуу өткөрүлүп берилет.


Game Launcher кызматын кандайча колдонгонуңуз тууралуу маалымат

Сиз берген маалыматтан тышкары, түзмөктөрүңүздөгү программалар же башка жолдор аркылуу Game Launcher кызматын кантип колдонгонуңуз тууралуу маалыматты да топтойбуз. Кандай маалымат топтолот:

• Game Launcher ишке киргизилген түзмөктөр, хештелген IMEI коду, MCC (мобилдик өлкө коду) жана MNC (мобилдик тармак коду) сыяктуу түзмөктүн окшоштугу жок өздүк билдиргичтери, ишке салуу системасынын версиялары жана Game Launcher менен Game Launcher'ден кем калбоо үчүн колдонулган түзмөктөрдүн орнотуулары;;
• Дарт аныктоо, техникалык, каталар жана керектөө маалыматы, анын ичинде Game Launcher кызматы кандайча, качан жана канчалык узак колдонулгандыгы, ошону менен катар Game Launcher кызматынын колдонууңуз тууралуу башка Game Launcher аркылуу сунушталган маңызы менен сиздин мамилелешкениңиз сыяктуу маалымат. Game Launcher'де же ал аркылуу ойнолгон оюндар жана биз сунуштаган оюндар тууралуу орнотулган жаратмалардын наамдары, сиз ойноп жаткан оюндун топтомунун аталышы, орнотулган жана жок кылынган оюндар сыяктуу маалыматты да топтойбуз; жана
• Өнөктөштөрдөн алынган маалымат: Өзүнчө бир оюнду ойногонуңузда, ачып же ойноп баштаган оюнду (оюндун версиясын да кошуп) канча жолу ачканыңыз же ойноп баштаганыңыз, ошондой эле өзүнчө бир оюнду канчалык узак ойногонуңуз тууралуу маалымат алабыз. Мындан тышкары, ойноп жаткан оюндардагы, окуу китебинин, эсептин же жетишкендиктин аягына чыкканыңыз, деңгээлдин аягына чыкканыңыздын жана деңгээлдин аягына чыга албаган убагы сыяктуу башкача окуяларды топтоп турабыз.

Сиз, түзмөктөрүңүз жана жаратмаларыңыз, ошондой эле Game Launcher'ди колдонууңуз тууралуу башка маалыматты, ал топтолуп жатканда сүрөттөлгөн жолдордо же сиздин макулдугуңуз менен башка жолдо топтошубуз мүмкүн.

Firebase Analytics сыяктуу үчүнчү тараптын талдоо кызматтарын колдонушубуз мүмкүн. Биз алган маалымат ушул кызмат тейлөөчүлөргө ачылышы мүмкүн же маалыматты колдонгон алдагы кызмат тейлөөчүлөр жана башка тийиштүү тараптар түзүнөн чогултушу мүмкүн, мисалы, Game Launcher'дин колдонуусун баалоого жана Game Launcher'ди башкаруу ишине жардам берүүгө. Firebase Analytics жөнүндө көбүрөөк маалымат алуу үчүн https://www.google.com/policies/privacy/partners/ дарегине баш багыңыз.





МААЛЫМАТЫҢЫЗ КАНДАЙЧА КОЛДОНУЛАТ?

Топтолгон маалымат төмөнкүдөй максаттарда колдонулат:

• Game Launcher'ди пайдаланууга жарамдуу кылыш үчүн сиздин өздүгүңүздү билүүгө жана накталыгыңызды аныктоо үчүн;
• Game Launcher менен сизди камсыздоо үчүн;
• Game Launcher менен башыңыздан өткөн жана кийинки аракеттерди камтыган тажрыйбаңызды жакшыртып жана башкачалатуу: боюңузга бычылган маңызды жана жоопторду жеткирүүсү жана Game Launcher менен мамилелешкен жол сыяктуусу (мисалы, кишилик жетишкендиктериңизди жана илгерилөөңүздү сактоо жана мурдакы иш-аракеттерге таянып мобилдик оюндар боюнча жекелештирилген сунуштоолор менен камсыз кылуу);
• Samsung Account’уңузду Samsung’дун же үчүнчү тараптардын түзмөктөрүнө, жаратмаларына, өзгөчөлүктөрүнө же кызматтарына байланыштыруу;
• Game Launcher аркылуу берген суроо-талаптарыңызга, сурап билүүлөргө жана көрсөтмөлөрүңүзгө жооп берүү;
• жарнамалоону, анын ичинде башкачалаткан жарнамаларды, демөөрчүнүн маңызы жана сүрөөгө алынган кабарлашуулар;
• катышуучу үчүнчү тараптын жаратмалары жана кызматтары менен өз ара мамилелешүүгө жана колдонууга жол берүү;
• Game Launcher жана компаниябызды ишке салып, баалап жана жакшыртуу үчүн (анын ичинде жаңы өнүмдөр менен кызматтарды өнүктүрүү; өнүмдөрүбүз менен кызматтарыбызды байытуу жана жакшыртуу; байланыштарыбызды тейлөө; өнүмдөрүбүздү, кызматтарыбызды жана кардарлар базасын талдоо; базар изилдөөсүн алып баруу; колдомолор талдоосун аткаруу; жана эсеп-кысап, аудит жана башка ички иш-милдеттерди аткаруу);
• алдамчылык сыяктуу кылмыш иштеринен, дооматтардан жана башка жоопкерчиликтерден коргоп, өздүгүн билип жана алдын алуу үчүн; жана
• жаратылуучу мыйзамдын талаптарына, тийиштүү өнөр-жайдын стандарттарына жана саясаттарыбызга, анын ичинде бул Жекелик эскертүүсү жана Жекелик саясаты да бар, көнүп жана аларды иш жүзүндө аткаруу.

Game Launcher'ден сиз жөнүндө чогултуп алынган маалыматты биз колдонуп, жана Galaxy Store/Samsung Free Play кыстырмасын колдонгонузда алынган сыяктуу башка кызматтардан, өзгөчөлүктөрдөн же сиз пайдаланып жүргөн түзмөктөрдөн алынган маалымат менен айкалыштырабыз, сиз Game Launcher жана Galaxy Store/Samsung Free экөөндөн алынган оюн керектөө жаздырууларызды айкалыштырган колдомолоруңуздун негизинде башкачалаткан кызматтарды ала аласыз.

Game Launcher башка вебсайттарга, жаратмаларга, кызматтарга же өзгөчөлүктөргө туташууга же аларды, Samsung же үчүнчү тараптын кызматтарынын жакшы көргөн же пайдаланып жүргөн маңызын көрсөткөндөй, көрсөтүү мүмкүнчүлүгүн бере алат. Үчүнчү тараптын вебсайттар, жаратмалар, кызматтар менен өзгөчөлүктөр бизге көз каранды эмес болгондуктан, алар өз алдынча ишке салынат жана өздөрүнүн жекелик эскертүүлөр же саясаттары болушу мүмкүн. Андыктан, алар сиздин маалыматыңызды кандайча иштете турганын билүү үчүн аларды кылдат окуп чыгыңыз. Башка вебсайт, колдонмо, кызмат же функция бизге таандык эмес болуп, биз көзөмөлдөбөгөндүктөн, алардын материалдарына, колдонулушуна жана купуялык саясаттарына жооп бербейбиз.




АККАУНТТУ ШИЛТЕЛӨӨСҮ

Айрым үчүнчү тараптардын кызматтары сизге Samsung account’уңузду үчүнчү тараптардын аккаунттары менен байланыштырууга жол берген аккаунттун шилтелөөсүн сунушташы мүмкүн.. Аккаунтуңуздар байланыштырылгандан кийин, үчүнчү тараптын кызматынын айрым өзгөчөлүктөрүн ошол үчүнчү тараптын жаратмасын орнотпостон пайдалана аласыз. Мисалы, Samsung account’’уңузга кирип, жаратылуучу кызматтагы шилтелөө тартибин аткарсаңыз, үчүнчү тарап сунуштаган үчүнчү тараптын иш-милдеттердин бир бөлүгүн колдонууга мүмкүнчүлүк аласыз.

Аккаунтуңуз шилтеленген сайын, айрым кишилик колдомолоруңуз алдагы үчүнчү тарапка жөнөтүлөт жана үчүнчү тарап ошол айрым кишилик колдомолорду Samsung көздөй жиберет. Аккаунтуңузду байланыштыруудан мурун, жеке маалыматтын кандай түрү кандай максатта бөлүшүлө тургандыгын эскертебиз.

Аккаунттар шилтеленгениңизде, аларды кантип ажыратууга боло турганын да кабарлап коебуз.




МААЛЫМАТЫҢЫЗДЫ КИМ МЕНЕН БӨЛҮШӨБҮЗ?

Маалыматыңызды өз компаниябыздын ичинде жана төмөнкү мекемелерге жогоруда келтирилген максаттарда гана беребиз.

• Бизнес өнөктөштөрүбүз: Game Launcher менен камсыздоого биз менен кызматташкан, Samsung account’уңузга өз аккаунттарын шилтелөөгө жол берген үчүнчү тараптын кызматтары сыяктуу өнөктөштөр. Бул бизнес өнөктөштөрү кишилик маалыматыңызды башкарып жана уюштурушат.
• Кызмат тейлөөчүлөр: биз үчүн колдомолорду сактаган булут платформаларды камсыз кылган компаниялар сыяктуу бизге же биздин атыбыздан кызмат көрсөтүүчү кылдат тандалган компаниялар. Бул кызмат көрсөтүүчүлөр да маалыматтарыңызды коргоого милдеттенет.
• Башка тараптар, мыйзам талап кылган учурда же Game Launcher түзмөгүңүздү коргоо үчүн: мисалы, мыйзам, сот иши же соттун чечими аркылуу өкмөт органдары маалыматыңызды ачыкка чыгарууну талап кылышы мүмкүн. Алар ошондой эле, укук коргоо, улуттук коопсуздук, терроризмге каршы күрөшүү же коомдук коопсуздукка байланышкан башка маселелер боюнча маалыматызды бизден сурашы мүмкүн;
• Корпоративдик транзакцияларга байланыштуу башка тараптарга: куюлуштуруу же өткөрүү, сатып алуу же сатуу бөлүгү катары же банкрот болгон учурда маалыматыңызды үчүнчү тарапка ачыкка чыгарышыбыз мүмкүн; жана
• Сиздин макулдугуңуз же талабыңыз боюнча башка тараптарга: Жекелик эскертүүсүндө баяндалган ачыкка чыгаруу учурларынан тышкары, өзүңүз макулдук бергенде же өтүнгөнүңүздө маалыматыңызды үчүнчү тараптар менен бөлүшүүбүз мүмкүн.




КУПУЯЛЫК ЭСКЕРТҮҮСҮНӨ КИРГИЗИЛЕ ТУРГАН ӨЗГӨРТҮҮЛӨР

Game Launcher кызматына байланыштуу маалыматты топтоп, иштетүү ыкмаларына киргизилген өзгөрүүлөрдү же тийиштүү мыйзамдардагы өзгөрүүлөрдү ушул Купуялык эскертүүсүн жаңыртып, кабарлап турабыз. Документтин акыркы жолу жаңыртылган күнү ушул Купуялык эскертүүсүнүн өйдө жагында көрүнөт.
