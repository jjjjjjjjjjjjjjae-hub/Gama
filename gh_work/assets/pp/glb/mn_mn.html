“Game Launcher” үйлчилгээний Хувийн нууцын мэдэгдэл


Хүчин төгөлдөр болох огноо: 2021 оны 8-р сарын 24

Samsung Electronics Co., Ltd. (“Samsung”) компанийн хамт олон бид манай хэрэглэгчдийн хувьд нууцлал ямар чухал болохыг мэддэг. Бид таны хувийн мэдээллийг хэрхэн цуглуулж, ашигладаг талаар танд ойлголт өгөхийн тулд энэхүү Хувийн нууцын мэдэгдлийг бий болгосон.

Манай https://account.samsung.com/membership/terms/privacypolicy холбоос дээрх Хувийн нууцын журмыг Game Launcher үйлчилгээг ашиглахад мөн мөрдөнө. Манай Нууцлалын бодлогод таны өгөгдлийг хэрхэн ашигладаг талаар илүү их мэдээлэл агуулагдана. Түүнчлэн бидэнтэй хэрхэн холбогдож болох тухай мэдээлэл багтсан. Энэхүү Нууцлалын мэдэгдлийн зэрэгцээ манай Нууцлалын бодлогыг уншина уу. Гэвч бид таны мэдээллийг Game Launcher үйлчилгээний зорилгоор хэрхэн ашиглах хүрээнд энэхүү Хувийн нууцын мэдэгдэл нь Хувийн нууцын журмаас ямагт давуу эрхтэй үйлчилнэ.

•Та Хэрэглэгчийн үйлчилгээнд хандах үед хариулт өгөхийн тулд бид таны хэн болохыг нягтлах мэдээлэл, бидэнтэй холбоо барих шалтгаан болсон бүтээгдэхүүний талаар асууна. •Таны манай Хэрэглэгчийн үйлчилгээний ажилтнууд эсхүл өөр бусадтай харилцсан харилцааг манай системээр дамжуулан хүргэнэ.

•Оношилгооны, техникийн, алдааны болон ашиглалтын мэдээлэл, тухайлбал, та Game Launcher-ыг хэрхэн, хэзээ, хэр удаан ашигладаг болох, Game Launcher-аар дамжин хүргэдэг контенттой хэрхэн харилцдаг зэрэг таны Game Launcher ашиглалттай холбоотой бусад мэдээлэл. Түүнчлэн бид таны Game Launcher дээр болон түүгээр дамжуулан тоглодог тоглоомуудын талаарх мэдээлэл, бидний танд санал болгосон тоглоомууд, суулгасан хэрэглүүрүүдийн нэр, таны тоглож буй тоглоомын багцын нэр болон суулгасан болон устгасан тоглоомуудын талаарх мэдээлэл зэргийг цуглуулна; мөн

Түүнчлэн бид таны тоглодог тоглоом дахь тусгай үйл явдлуудын талаарх мэдээлэл, тухайлбал зааварлагч үеийг дуусгасан тухай, оноо эсвэл амжилт, аль нэг үеийг дуусгах эсхүл давж чадаагүй талаарх мэдээлэл цуглуулна.

Мөн та, таны төхөөрөмж, аппын тухай, түүнчлэн таны Game Launcher-ыг хэрхэн ашиглаж буй талаарх бусад мэдээллийг мэдээлэл цуглуулах үед танд тайлбарлаж өгсөн арга, танаас зөвшөөрлөөр өөр бусад аргаар тус тус цуглуулж болно.

Бид Firebase Analytics зэрэг гуравдагч этгээдийн дүн шинжилгээний үйлчилгээг ашиглаж болно. Бидний авсан мэдээллийг эдгээр үйлчилгээ үзүүлэгчдэд болон тухайн мэдээллийг, жишээлбэл, Game Launcher ашиглалтыг үнэлэх болон Game Launcher үйлчилгээг хүргэх зорилгоор, бусад холбогдох гуравдагч этгээдэд задруулах буюу тэд өөрсдөө шууд цуглуулах боломжтой. Тэд таны мэдээллийг хэрхэн боловсруулдаг болохыг ойлгохын тулд үүнийг уншиж танилцана уу. БҮРТГЭЛ ХОЛБОХ

Зарим гуравдагч этгээдийн үйлчилгээ нь таны Samsung аccount-ыг гуравдагч этгээдийн үйлчилгээний бүртгэлүүдтэй холбох бүртгэлийн холболтыг санал болгож болно. Таны бүртгэлүүд холбогдмогц та тухайн гуравдагч этгээдийн хэрэглүүрийг суулгах шаардлагагүйгээр гуравдагч этгээдийн үйлчилгээний зарим үзүүлэлтүүдийг хэрэглэх боломжтой болно. Жишээ нь та Samsung аccount-д нэвтэрч холбогдох үйлчилгээ дэх холбох үйл явцын дуусгах үед гуравдагч этгээдээс санал болгож буй тухайн гуравдагч этгээдийн функцуудыг хэрэглэх боломжтой болно.

Бүртгэлийг холбох тухай бүрд тухайн гуравдагч этгээдэд зарим хувийн өгөгдлийг илгээж, тухайн гуравдагч этгээд хувийн мэдээллийг Samsung-д мөн илгээнэ. Холбохоос өмнө бид танд хуваалцах Хувийн өгөгдлийн талаар танд тусгайлан мэдэгдэж, ямар зорилгоор хувийн мэдээллийг хуваалцахыг мэдэгдэнэ.

Та бүртгэлүүдийг холбоход бид тэдгээр холбосон бүртгэлүүдээ хэрхэн салгах талаар мөн мэдээлэл өгнө.
