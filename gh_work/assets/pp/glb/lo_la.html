ແຈ້ງການຄວາມເປັນສ່ວນຕົວຂອງ Game Launcher


ວັນທີມີຜົນບັງຄັບໃຊ້: 24 ສິງຫາ 2021

ທີ່ Samsung Electronics Co., Ltd. (“Samsung”), ພວກເຮົາຮູ້ດີວ່າ ຄວາມເປັນສ່ວນຕົວ ມີຄວາມສຳຄັນແນວໃດສຳລັບ ລູກຄ້າ ຂອງພວກເຮົາ. ພວກເຮົາໄດ້ສ້າງ ແຈ້ງການ ກ່ຽວກັບ ຄວາມເປັນສ່ວນຕົວ ນີ້ຂຶ້ນມາ ເພື່ອໃຫ້ແນ່ໃຈວ່າທ່ານເຂົ້າໃຈວ່າພວກເຮົາເກັບກຳແລະນຳໃຊ້ຂໍ້ມູນ ສ່ວນຕົວ ຂອງທ່ານແນວໃດແນ່.

ນະ​ໂຍ​ບາຍ ຄວາມເປັນສ່ວນຕົວ​ ຂອງ​ພວກເຮົາ​ທີ່ https://account.samsung.com/membership/terms/privacypolicy ຍັງ​ນຳ​ໃຊ້​ກັບ​ການ​​ໃຊ້​ Game Launcher. ນະ​ໂຍ​ບາຍ​ ຄວາມເປັນສ່ວນຕົວ ​ຂອງ​ພວກ​ເຮົາ​ມີ​ຂໍ້​ມູນ ​ເພີ່ມເຕີມ ​ກ່ຽວ​ກັບ​ວ່າ​ພວກ​ເຮົາ​ໃຊ້ ​ຂໍ້ມູນ ​ຂອງ​ທ່ານ​ແນວ​ໃດແນ່. ນອກ​ນັ້ນ​ມັນ​ຍັງ​ມີ​ຂໍ້​ມູນ​ກ່ຽວ​ກັບ​ວິ​ທີ​ຕິດ​ຕໍ່​ຫາພວກ​ເຮົາ​ນຳ​ອີກ. ກະ​ລຸ​ນາອ່ານ​ ນະ​ໂຍ​ບາຍ​ ຄວາມ​ເປັນ​ສ່ວນ​ຕົວ​ຂອງ​ພວກ​ເຮົາ​, ນອກເໜືອຈາກ​ແຈ້ງ​ການຄວາມ​ເປັນ​ສ່ວນ​ຕົວ​ນີ້. ເຖິງ​ແນວ​ໃດ​ກໍ່​ຕາມ, ແຈ້ງການ​ຄວາມ​ເປັນ​ສ່ວນ​ຕົວ​ນີ້​ຈະ​ມີ​ຜົນ​ເໜືອກວ່າ​ນະ​ໂຍ​ບາຍ​ຄວາມ​ເປັນ​ສ່ວນ​ຕົວທີ່ກ່ຽວຂ້ອງກັບ​ກັບ​ວິ​ທີ​ທີ່​ພວກ​ເຮົາ​ໃຊ້​ຂໍ້ມູນຂອງທ່ານສຳລັບ Game Launcher.




ພວກເຮົາເກັບເອົາຂໍ້ມູນອັນໃດແນ່?

ຜ່ານ Game Launcher, Samsung ຈະໄດ້ຮັບແລະຮັກສາຂໍ້ມູນກ່ຽວກັບທ່ານດ້ວຍຫຼາຍວິທີ.

ຂໍ້​ມູນ​ທີ່​ທ່ານ​ສະໜອງໃຫ້​ໂດຍ​ກົງ

•ເມື່ອທ່ານ ລົງຊື່ເຂົ້າໃຊ້ Samsung Account, ຂໍ້ມູນ Samsung Account ເຊັ່ນວ່າ Samsung account ID, ຊື່ຫຼິ້ນ, ຮູບ ໂປຣໄຟລ໌, ອາຍຸ ແລະ ປະເທດ; ແລະ
•ເພື່ອ​ໃຫ້​ຄຳ​ຕອບ​ເມື່ອ​ທ່ານ​ຕິດ​ຕໍ່​ຝ່າຍ​ບໍ​ລິ​ການ​ລູກ​ຄ້າ, ພວກ​ເຮົາ​ຖາມ​ເອົາ​ຂໍ້​ມູນ​ເພື່ອກວດ​ສ​ອບ​ຂໍ້​ມູນ​ປະ​ຈຳ​ຕົວ​ຂອງ​ທ່ານແລະກ່ຽວກັບຜະ​ລິດ​ຕະ​ພັນ​ທີ່​ທ່ານ​ກຳ​ລັງ​ຕິດ​ຕໍ່​ພວກ​ເຮົາຢູ່​. ການ​ສື່​ສານຂອງທ່ານ​ກັບ​ພະ​ນັກ​ງານ​ບໍ​ລິ​ການ​ລູກ​ຄ້າ​ ຫຼື ຜູ້​ອື່ນໆຈະ​ຖືກ​ສົ່ງ​ຜ່ານ​ລະ​ບົບ​ຂອງ​ພວກ​ເຮົາ.


ຂໍ້ມູນການນຳໃຊ້ Game Launcher ຂອງທ່ານ

ນອກຈາກຂໍ້ມູນທີ່ທ່ານໃຫ້ແລ້ວ, ພວກເຮົາຈະ ສະສົມ ເກັບກຳຂໍ້ມູນການໃຊ້ Game Launcher ຂອງພວກ​ເຮົາຜ່ານຊອບແວໃນ ອຸປະກອນ ຂອງທ່ານແລະດ້ວຍວິທີການອື່ນໆ. ພວກເຮົາຈະເກັບເອົາ:

•ຂໍ້ມູນກ່ຽວກັບ ອຸປະກອນ ທີ່ເປີດໃຊ້ງານ Game Launcher ຂອງທ່ານ, ຕົວລະບຸ ອຸປະກອນ ສະເພາະເຊັ່ນ: IMEI, MCC ທີ່ຖືກແຮຊ໌ (hashed) ແລ້ວ (ລະຫັດປະເທດຂອງມືຖື), ແລະ MNC (ລະຫັດເຄືອຂ່າຍມືຖື), ເວີຊັນຂອງ ລະ​ບົບ ປະຕິບັດການແລະ ການຕັ້ງຄ່າ ຂອງ Game Launcher ແລະອຸປະກອນຕ່າງໆທ່ານໃຊ້ເຂົ້າເຖິງ Game Launcher;

•​ ຝ່າຍອື່ນໆທີ່ທ່ານໄດ້ໃຫ້​ຄຳ​ຍິນຍອມ ຫຼື ໃນຄຳ​ແນະ​ນຳຂອງທ່ານ: ນອກຈາກ​ການ​ເປີດ​ເຜີຍ​ທີ່​ໄດ້ອະທິບາຍ​ໄວ້​ໃນແຈ້ງ​ການຄວາມເປັນສ່ວນຕົວນີ້​ແລ້ວ, ພວກ​ເຮົາ​ອາດ​ຈະແຊ​ຣ໌​ຂໍ້​ມູນກ່ຽວກັບທ່ານ​ໃຫ້​ກັບ​ພາກ​ສ່ວນທີ​ສາມທີ່​ທ່ານອະນຸຍາດໄວ້ຕ່າງຫາກ ຫຼື ໄດ້ຂໍ​ໃຫ້​ມີ​ການ​ແຊ​ຣ໌​ດັ່ງກ່າວ.




ການອັບ​ເດດແຈ້ງການຄວາມເປັນສ່ວນຕົວນີ້

​ແຈ້ງ​ການ​ຄວາມ​ເປັນ​ສ່ວນ​ຕົວ​ນີ້​ອາດ​ຈະ​ຖືກ​ອັບ​ເດດ​ເພື່ອໃຫ້​ທ່ານ​ຮູ້​ກ່ຽວ​ກັບ​ການ​ປ່ຽນ​ແປງວິ​ທີ​ການ​ທີ່​ພວກ​ເຮົາ​ເກັບ​ເອົາ ແລະ ປະ​ມວນ​ຜົນ​ຂໍ້​ມູນ​ຂອງ​ທ່ານຢູ່​ໃນ Game Launcher ຫຼື ການ​ປ່ຽນ​ແປງໃນກົດ​ໝາຍ​ທີ່​ກ່ຽວ​ຂ້ອງ. ວັນ​ທີ​ເມື່ອ​ເອ​ກະ​ສານ​ຖືກ​ອັບ​ເດດ​ຫຼ້າ​ສຸດ​ສະ​ແດງ​ຂຶ້ນ​ຢູ່​ດ້ານ​ເທິງ​ສຸດ​ຂອງ​ແຈ້ງ​ການ​ຄວາມ​ເປັນ​ສ່ວນ​ຕົວ​ນີ້.
