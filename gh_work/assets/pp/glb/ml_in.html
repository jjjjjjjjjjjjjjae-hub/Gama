Game Launcher - സ്വകാര്യതാ അറിയിപ്പ്


പ്രാബല്യ തീയതി: 24 ആഗസ്‌റ്റ് 2021

Samsung Electronics Co., Ltd.-ന് (“Samsung”) എത്രത്തോളം പ്രധാനമാണ് ഉപഭോക്താക്കൾക്ക് സ്വകാര്യതയെന്ന് അറിയാം. നിങ്ങളുടെ വ്യക്തിഗത വിവരങ്ങൾ ഞങ്ങൾ എങ്ങനെയാണ് ശേഖരിക്കുകയും ഉപയോഗിക്കുകയും ചെയ്യുന്നതെന്ന് നിങ്ങൾ മനസ്സിലാക്കുന്നുവെന്ന് ഉറപ്പാക്കുന്നതിനാണ് ഞങ്ങൾ ഈ സ്വകാര്യതാ അറിയിപ്പ് സൃഷ്ടിച്ചിരിക്കുന്നത്.

https://account.samsung.com/membership/terms/privacypolicy എന്ന വിലാസത്തിലുള്ള ഞങ്ങളുടെ സ്വകാര്യതാ നയവും നിങ്ങളുടെ Game Launcher ഉപയോഗത്തിന് ബാധകമാണ്. നിങ്ങളുടെ ഡാറ്റ ഞങ്ങൾ എങ്ങനെയാണ് ഉപയോഗിക്കുന്നത് എന്നതിനെ കുറിച്ചുള്ള കൂടുതൽ വിവരങ്ങൾ ഞങ്ങളുടെ സ്വകാര്യതാ നയത്തിൽ അടങ്ങിയിരിക്കുന്നു. ഞങ്ങളെ എങ്ങനെ ബന്ധപ്പെടാം എന്നതിനെ കുറിച്ചുള്ള വിവരങ്ങളും ഇതിൽ ഉൾപ്പെടുന്നു. ഈ സ്വകാര്യതാ അറിയിപ്പിന് പുറമേ, ഞങ്ങളുടെ സ്വകാര്യതാ നയവും വായിക്കുക. എന്നിരുന്നാലും, നിങ്ങളുടെ വിവരങ്ങൾ Game Launcher-നായി ഞങ്ങൾ എങ്ങനെ ഉപയോഗിക്കുന്നു എന്നതുമായി ബന്ധപ്പെട്ട് സ്വകാര്യതാ നയത്തെക്കാൾ ഈ സ്വകാര്യതാ അറിയിപ്പിനായിരിക്കും എല്ലായ്പ്പോഴും മുൻഗണന.




എന്തൊക്കെ വിവരങ്ങളാണ് ഞങ്ങൾ ശേഖരിക്കുന്നത്?

Game Launcher-ലൂടെ, നിങ്ങളെ കുറിച്ചുള്ള വിവരങ്ങൾ വ്യത്യസ്ത വഴികളിലൂടെ Samsung കരസ്ഥമാക്കുകയും പരിപാലിക്കുകയും ചെയ്യുന്നു.

നിങ്ങൾ നേരിട്ട് നൽകുന്ന വിവരങ്ങൾ

• നിങ്ങൾ Samsung Account-ലേക്ക് ലോഗ് ഇൻ ചെയ്യുമ്പോൾ, നിങ്ങളുടെ Samsung Account ID, വിളിപ്പേര്, പ്രൊഫൈൽ ചിത്രം, പ്രായം, രാജ്യം എന്നിവ പോലുള്ള Samsung Account വിവരങ്ങൾ; ഒപ്പം
• നിങ്ങൾ ഉപഭോക്തൃ സേവനവുമായി ബന്ധപ്പെടുമ്പോൾ നിങ്ങൾക്ക് പ്രതികരണങ്ങൾ നൽകുന്നതിന്, നിങ്ങളുടെ ഐഡന്റിറ്റിയും ഏത് ഉൽപ്പന്നവുമായി ബന്ധപ്പെട്ടാണോ നിങ്ങൾ ഞങ്ങളെ ബന്ധപ്പെടുന്നത് ആ ഉൽപ്പന്നവും പരിശോധിച്ചുറപ്പിക്കുന്നതിന് ഞങ്ങൾ വിവരങ്ങൾ ആവശ്യപ്പെടും. ഞങ്ങളുടെ ഉപഭോക്തൃ സേവന സ്റ്റാഫുമായുള്ള അല്ലെങ്കിൽ മറ്റുള്ളവരുമായുള്ള നിങ്ങളുടെ ആശയവിനിമയം ഞങ്ങളുടെ സിസ്റ്റങ്ങളിലൂടെ കൈമാറ്റം ചെയ്യപ്പെടും.


നിങ്ങളുടെ Game Launcher ഉപയോഗത്തെ കുറിച്ചുള്ള വിവരങ്ങൾ

നിങ്ങൾ നൽകുന്ന വിവരങ്ങൾക്ക് പുറമേ, നിങ്ങളുടെ ഉപകരണങ്ങളിലെ സോഫ്‌റ്റ്‌വെയർ വഴിയും മറ്റ് മാർഗ്ഗങ്ങളിലൂടെയും നിങ്ങളുടെ Game Launcher ഉപയോഗത്തെ കുറിച്ചുള്ള വിവരങ്ങൾ ഞങ്ങൾ ശേഖരിക്കും. ഞങ്ങൾ ഇനിപ്പറയുന്നവ ശേഖരിക്കും:

• Game Launcher പ്രാപ്‌തമാക്കിയിട്ടുള്ള നിങ്ങളുടെ ഉപകരണങ്ങളെ കുറിച്ചുള്ള വിവരങ്ങൾ, ഹാഷ് ചെയ്ത IMEI-യും MCC-യും (മൊബൈൽ രാജ്യ കോഡ്), MNC-യും (മൊബൈൽ നെറ്റ്‌വർക്ക് കോഡ്), ഓപ്പറേറ്റിംഗ് സിസ്റ്റം പതിപ്പുകളും പോലുള്ള യുണീക് ഉപകരണ ഐഡന്റിഫയറുകൾ, Game Launcher-ന്റെയും Game Launcher ആക്‌സസ് ചെയ്യാൻ നിങ്ങൾ ഉപയോഗിക്കുന്ന ഉപകരണങ്ങളുടെയും സജ്ജീകരണം;
• Game Launcher വഴി നിങ്ങൾക്ക് നൽകപ്പെടുന്ന ഉള്ളടക്കവുമായി നിങ്ങൾ എങ്ങനെ ഇന്റരാക്‌റ്റ് ചെയ്യുന്നു എന്നതുപോലെ Game Launcher-ന്റെ ഉപയോഗത്തെ കുറിച്ചുള്ള മറ്റ് വിവരങ്ങൾക്കൊപ്പം, നിങ്ങൾ എങ്ങനെയാണ്, എപ്പോഴാണ്, എത്ര സമയമാണ് Game Launcher ഉപയോഗിക്കുന്നത് എന്നത് ഉൾപ്പെടെയുള്ള ഡയഗ്നോസ്റ്റിക് വിവരങ്ങളും സാങ്കേതിക വിവരങ്ങളും പിശക് വിവരങ്ങളും ഉപയോഗ വിവരങ്ങളും. Game Launcher-ലോ അതിലൂടെയോ നിങ്ങൾ കളിക്കുന്ന ഗെയിമുകളെ കുറിച്ച്, ഇൻസ്റ്റാൾ ചെയ്ത ആപ്ലിക്കേഷനുകളുടെ പേരുകളും നിങ്ങൾ കളിക്കുന്ന ഗെയിമിന്റെ പാക്കേജിന്റെ പേരും ഇൻസ്റ്റാൾ ചെയ്തതും ഇല്ലാതാക്കിയതുമായ ഗെയിമുകളും പോലുള്ള വിവരങ്ങൾ ഞങ്ങൾ ശേഖരിക്കുന്നു; ഒപ്പം
• പങ്കാളികളിൽ നിന്നുള്ള വിവരങ്ങൾ: നിങ്ങളൊരു പ്രത്യേക ഗെയിം കളിക്കുമ്പോൾ, നിങ്ങൾ തുറന്നതോ കളിക്കാൻ തുടങ്ങിയതോ ആയ ഗെയിമിനെ കുറിച്ചുള്ള വിവരങ്ങളും (ഗെയിമിന്റെ പതിപ്പ് ഉൾപ്പെടെ), നിങ്ങൾ എത്ര തവണ ഒരു പ്രത്യേക ഗെയിം തുറക്കുകയും അല്ലെങ്കിൽ കളിക്കാൻ തുടങ്ങുകയും എത്രനേരം കളിക്കുകയും ചെയ്യുന്നു എന്ന വിവരങ്ങളും. ഇതിന് പുറമെ, നിങ്ങൾ കളിക്കുന്ന ഗെയിമുകളിലെ ഇച്ഛാനുസൃത ഇവന്റുകൾ ഞങ്ങൾ ശേഖരിക്കും, അതായത് നിങ്ങൾ എപ്പോഴാണ് ഒരു ട്യൂട്ടോറിയൽ പൂർത്തിയാക്കുന്നത്, സ്‌കോറോ നേട്ടമോ കൈവരിക്കുന്നത്, ഒരു ലെവൽ പൂർത്തിയാക്കുമ്പോൾ, ഒരു ലെവലിൽ പരാജയപ്പെടുന്നത് എന്നിങ്ങനെയുള്ള വിവരങ്ങൾ.

ശേഖരണ സമയത്ത് ഞങ്ങൾ വിവരിക്കുന്ന മാർഗ്ഗങ്ങളിലൂടെയോ, അല്ലെങ്കിൽ നിങ്ങളുടെ സമ്മതത്തോടെയോ, നിങ്ങളെ കുറിച്ചും നിങ്ങളുടെ ഉപകരണങ്ങളെ കുറിച്ചും ആപ്ലിക്കേഷനുകളെ കുറിച്ചും Game Launcher-ന്റെ നിങ്ങളുടെ ഉപയോഗത്തെ കുറിച്ചുമുള്ള മറ്റ് വിവരങ്ങളും ഞങ്ങൾ ശേഖരിച്ചേക്കാം.

Firebase Analytics പോലുള്ള മൂന്നാം കക്ഷി അനലിറ്റിക്‌സ് സേവനങ്ങൾ ഞങ്ങൾ ഉപയോഗിച്ചേക്കാം. ഞങ്ങൾ കരസ്ഥമാക്കുന്ന വിവരങ്ങൾ, ഈ ദാതാക്കൾക്കും മറ്റ് പ്രസക്തമായ മൂന്നാം കക്ഷികൾക്കും നേരിട്ട് വെളിപ്പെടുത്തുകയോ അവരത് നേരിട്ട് ശേഖരിക്കുകയോ ചെയ്യാം, ഉദാഹരണത്തിന്, Game Launcher-ന്റെ ഉപയോഗം വിലയിരുത്തുന്നതിനും Game Launcher-മായി ബന്ധപ്പെട്ട നിർവഹണ സംവിധാനങ്ങളിൽ സഹായിക്കുന്നതിനുമാണ് ഈ വിവരങ്ങൾ ഉപയോഗിക്കപ്പെടുക.
