Databeskyttelseserklæring for Game Launcher


Ikrafttrædelsesdato: 24. august 2021

Hos Samsung Electronics Co., Ltd. (“Samsung”) ved vi, hvor vigtigt privatlivets fred er for vores kunder. Samsung er den dataansvarlige for Game Launcher. Vi har oprettet denne databeskyttelseserklæring for at sikre, at du forstår, hvordan vi indsamler og bruger dine personoplysninger.

Vores databeskyttelsespolitik på https://account.samsung.com/membership/terms/privacypolicy gælder også for din brug af Game Launcher. Vores databeskyttelsespolitik indeholder flere oplysninger om, hvordan vi bruger dine oplysninger. Den indeholder også oplysninger om dine rettigheder, og hvordan du kan kontakte os. Du bedes læse vores databeskyttelsespolitik i tillæg til denne databeskyttelseserklæring. Denne databeskyttelseserklæring skal dog altid have forrang over databeskyttelsespolitikken i forhold til, hvordan vi bruger dine oplysninger til Game Launcher.




HVILKE OPLYSNINGER INDSAMLER VI?

Gennem Game Launcher, indsamler og opretholder Samsung oplysninger om dig på forskellige måder.

Oplysninger, som du tilvejebringer direkte

• Når du logger på din Samsung account, dine Samsung account-oplysninger, som for eksempel dit Samsung account-id, kaldenavn, profilbillede, alder, og land, og
• For at svare dig, når du kontakter kundeservice, beder vi om oplysninger for at bekræfte din identitet og det produkt, du kontakter os om. Din kommunikation med vores kundeservicemedarbejdere eller andre sendes gennem vores systemer.

Oplysninger om din brug af Game Launcher

Ud over de oplysninger, som du tilvejebringer, indsamler vi oplysninger om din brug af vores Game Launcher gennem software på dine enheder og via andre midler. Vi indsamler:

• Oplysninger om dine Game Launcher-aktiverede enheder, unikke enhedsidentifikatorer såsom krypteret IMEI, MCC (mobil landekode), og MNC (mobilnetværkskode), operativsystemversioner og indstillinger for Game Launcher og de enheder, som du bruger til at få adgang til Game Launcher,
• Diagnostiske, tekniske, fejl- og brugsoplysninger, herunder om hvordan, hvornår og hvor længe du bruger Game Launcher sammen med andre oplysninger om din brug af Game Launcher, såsom hvordan du interagerer med indhold, der tilbydes via Game Launcher. Vi indsamler også oplysninger om spil, som vi anbefaler til dig og de spil, du spiller på eller gennem Game Launcher, såsom titlerne på installerede apps, pakkenavnet på det spil, du spiller, og installerede og slettede spil, og
• Oplysninger fra partnere, Når du spiller et bestemt spil, modtager vi oplysninger om det spil, du åbnede eller begyndte at spille (herunder versionen af spillet), hvor mange gange du åbner eller begynder at spille spillet, og hvor længe du spiller et bestemt spil. Derudover indsamler vi tilpassede begivenheder i de spil, du spiller, såsom når du gennemfører et selvstudium, opnår en score eller præstation, når du gennemfører et niveau, og når det mislykkes for dig at gennemføre et niveau.

Vi kan også indsamle andre oplysninger om dig, dine enheder og apps, samt din brug af Game Launcher på måder, som vi beskriver for dig på indsamlingspunktet eller på anden måde med dit samtykke.

Vi kan bruge tredjepartsanalysetjenester, såsom Firebase Analytics. De oplysninger, som vi indsamler, kan videregives til eller indsamles direkte af disse udbydere og andre relevante tredjeparter, som bruger oplysningerne, for eksempel, til at evaluere brug af Game Launcher og til at administrere Game Launcher. Hvis du vil vide mere om Firebase Analytics, bedes du besøge https://www.google.com/policies/privacy/partners/.



HVORDAN BRUGER VI DINE OPLYSNINGER?
Vi bruger de indsamlede oplysninger til følgende formål:

• identificere og autentificere dig, så du kan bruge Game Launcher,
• levere Game Launcher til dig,
• forbedre og tilpasse din oplevelse med Game Launcher, såsom at levere indhold og svar, der er skræddersyet til dig og den måde, hvorpå du interagerer med Game Launcher (f.eks. gemme dine personlige præstationer og fremskridt) og levere indholdsanbefalinger,
• linke din Samsung account til forbundne Samsung- eller tredjepartsenheder, -apps, -funktioner eller -tjenester,
• svare på dine anmodninger, spørgsmål og anvisninger, der fremsættes gennem Game Launcher,
• levere annoncering, herunder tilpassede annoncer, sponsoreret indhold og salgsfremmende meddelelser (såsom via push-meddelelser),
• give dig mulighed for at interagere med og bruge visse tredjepartsapps og -tjenester,
• til at drive, evaluere og forbedre Game Launcher og vores virksomhed (herunder udvikling af nye produkter og tjenester, forbedring af vores produkter og tjenester, administration af vores meddelelser, analyse af vores produkter, tjenester og kundebase, udførelse af markedsundersøgelser, udførelse af dataanalyse og udførelse af regnskabsføring, revision og andre interne funktioner),
• beskytte mod, identificere og forebygge svig og anden kriminel aktivitet, krav og andre finansielle forpligtelser, og
• overholde og håndhæve gældende lovkrav, relevante branchestandarder og vores politikker, herunder denne databeskyttelseserklæring og databeskyttelsespolitikken.

Game Launcher kan give mulighed for at oprette forbindelse til eller vise andre websteder, apps, tjenester og andre funktioner, såsom at vise indhold, du kan lide eller bruger fra Samsung eller tredjepartstjenester. Tredjepartswebsteder, -apps, -tjenester og -funktioner fungerer muligvis uafhængigt af os og kan have deres egne databeskyttelseserklæringer eller -politikker. Gennemgå dem for at sikre, at du forstår, hvordan de behandler dine oplysninger. I det omfang, at et eventuelt linket eller vist websted, app, tjeneste eller anden funktion ikke ejes eller kontrolleres af os, er vi ikke ansvarlige for dets indhold, brug eller databeskyttelsespraksisser.

FORBINDELSE AF KONTI

Nogle tredjepartstjenester kan tilbyde muligheden for at forbinde konti, hvilket giver dig mulighed for at forbinde din Samsung account med konti hos tredjepartstjenester. Når dine konti er forbundet, kan du muligvis bruge nogle af funktionerne fra den pågældende tredjepartstjeneste uden installation af den pågældende tredjeparts applikation. Hvis du for eksempel logger ind på din Samsung account og gennemfører forbindelsesproceduren i den relevante tjeneste, kan du muligvis bruge en del af den pågældende tredjeparts funktioner, der tilbydes af en tredjepart.

Hver gang en konto forbindes, sendes visse personoplysninger til den pågældende tredjepart, og den pågældende tredjepart sender visse personoplysninger til Samsung. Forud for forbindelse af konti, informerer vi dig specifikt om, hvilke personoplysninger der vil blive delt, og til hvilket formål de specifikke personoplysninger vil blive delt.

Når du forbinder konti, informerer vi dig også om, hvordan du fjerner forbindelsen til konti.

Samsung behandler personoplysninger til de formål, der er beskrevet ovenfor. Samsungs retsgrundlag for behandling af personoplysninger er beskrevet nedenfor:

 Holde vores løfte til dig

Den generelle forordning om databeskyttelse Artikel 6(1)(b)

   • Identificere og autentificere dig

   • Levere Game Launcher til dig, såsom at give dig mulighed for at spille forskellige spil på eller gennem Game Launcher

   • Give dig mulighed for at interagere med andet Samsung- og tredjepartsindhold

 

For at fremme vores forretningsinteresser

Den generelle forordning om databeskyttelse Artikel 6(1)(f)

   • Drive, evaluere og forbedre Game Launcher og vores virksomhed

   • Levere tilpasset indhold til dig, såsom at gemme dine personlige præstationer og status

   • Levere indholdsanbefalinger til dig

   • Svare på dine anmodninger, anvisninger og forespørgsler

   • Opretholde passende sikkerhedsforanstaltninger

   • Beskytte mod erstatningsansvar, herunder overholdelse af branchestandarder og håndhævelse af vores politikker

For at overholde loven og retsprocesser

Den generelle forordning om databeskyttelse Artikel 6(1)(c)

   • Overholde gældende love og forskriftsmæssige påbud og retskendelser

 Med dit samtykke

Den generelle forordning om databeskyttelse Artikel 6(1)(a)

   • Levere markedsføringsmeddelelser (f.eks. via push-meddelelser)

   • Give dig mulighed for at forbinde din Samsung account med tredjepartsindhold

   • Vise dig interessebaserede annoncer

 





HVEM DELER VI DINE OPLYSNINGER MED?

Vi videregiver dine oplysninger internt i vores virksomhed og til følgende selskaber, men kun til de formål, der er beskrevet ovenfor.

• Forretningspartnere: partnere, som vi samarbejder med for at levere Game Launcher til dig, såsom tredjepartstjenester, der giver dig mulighed for at forbinde deres konto til din Samsung account. Disse forretningspartnere kontrollerer og administrerer dine personoplysninger.
• Tjenesteudbydere: omhyggeligt udvalgte selskaber, der leverer tjenester for eller på vegne af os, såsom cloud-platforme, der opretholder oplysninger på vores vegne. Disse udbydere er også forpligtede til at beskytte dine oplysninger.
• Andre parter, når det er påbudt ved lov eller som nødvendigt for at beskytte Game Launcher: det kan for eksempel være nødvendigt ved lov, retsproces eller retskendelse fra offentlige myndigheder at videregive dine oplysninger. De kan også anmode om dine oplysninger fra os med henblik på retshåndhævelse, national sikkerhed, bekæmpelse af terrorisme eller andre problemstillinger relateret til offentlig sikkerhed,
• Andre parter i forbindelse med virksomhedstransaktioner: vi kan videregive dine oplysninger til en tredjepart som en del af en fusion, overdragelse, opkøb eller salg, eller i tilfælde af konkurs, og
• Andre parter med dit samtykke eller efter dine anvisninger: ud over de videregivelser, der er beskrevet i denne databeskyttelseserklæring, kan vi dele oplysninger om dig med tredjeparter, når du giver særskilt samtykke til eller anmoder om en sådan deling.



HVOR LÆNGE OPBEVARER VI DINE OPLYSNINGER, OG HVOR SENDES DE HEN?

Hvor længe dine oplysninger opbevares afhænger af det retsgrundlag, der påberåbes til behandling af dine oplysninger. For eksempel kræver oplysninger, som vi behandler for at opfylde vores kontrakt med dig, at vi opbevarer oplysningerne i hele den tidsperiode, hvor du bruger Game Launcher. Så længe du er en aktiv bruger af Game Launcher, opbevarer og behandler vi disse oplysninger. Oplysninger, såsom dine kontooplysninger og enhedsoplysninger og -identifikatorer, hører ind under denne kategori.

Oplysninger, som vi behandler for at fremme vores forretningsinteresser, såsom din brug af Game Launcher, opbevares kun så længe som nødvendigt til de formål, hvortil de blev indsamlet. Disse oplysninger kan f.eks. indsamles med henblik på at udføre analyser, så vi kan udvikle forbedringer af Game Launcher, eller vi kan behandle oplysninger for at føre en optegnelse over forespørgsler, som du har foretaget gennem eller om Game Launcher, for at forbedre din kundeserviceoplevelse.

Derudover kan vi også behandle dine oplysninger baseret på dit samtykke. Hvis du trækker dit samtykke tilbage, fortsætter vi med at behandle oplysninger, der blev indsamlet forud for din tilbagetrækning, indtil de formål, hvortil sådanne oplysninger blev indsamlet, er opnået.

Bemærk venligst, selvom vi stræber efter at opbevare dine oplysninger i den ovenfor beskrevne periode, kan dine oplysninger muligvis behandles længere i medfør af gældende lovgivning. Hvis for eksempel en specifik lov foreskriver, at vi skal bruge en bestemt oplysning, overholder vi loven og opbevarer den pågældende oplysning, indtil den påkrævede opbevaringsperiode udløber.

Så længe dine oplysninger opbevares af os, vil dine oplysninger altid være underlagt passende sikkerhedsforanstaltninger.

Din brug af Game Launcher involverer overførsel, opbevaring og behandling af dine personoplysninger i andre lande; sådanne lande omfatter, uden begrænsning, lande i Det Europæiske Økonomiske Samarbejdsområde, Sydkorea og USA. Alle internationale dataoverførsler er underlagt lovkrav for at sikre, at dine personoplysninger behandles sikkert og som du forventer. Sådanne foranstaltninger omfatter brugen af standardkontraktbestemmelser til at beskytte overførslen af oplysninger uden for EØS. For at anmode om flere oplysninger eller få en kopi af de indgåede aftaler, bedes du se afsnittet KONTAKT OS i denne databeskyttelseserklæring.




HVORDAN BESKYTTER VI DINE OPLYSNINGER?

Vi tager databeskyttelse meget alvorligt. Vi har indført fysiske og tekniske sikkerhedsforanstaltninger til at beskytte de oplysninger, vi indsamler. Men vær opmærksom på, at selvom vi træffer rimelige foranstaltninger til at beskytte dine oplysninger, er der ikke noget websted, nogen internetoverførsel, computersystem eller trådløs forbindelse, der er fuldstændig sikker.




HVAD ER DINE RETTIGHEDER?

Dine personoplysninger tilhører dig. Du kan bede os om at tilvejebringe nærmere oplysninger om, hvad vi har indsamlet, og du kan bede os om at slette dem eller berigtige eventuelle unøjagtigheder. Du kan også bede os om at begrænse behandlingen, delingen eller overførslen af dine personoplysninger, samt at stille de personoplysninger, som vi har indsamlet om dig, til rådighed for dig, så du kan bruge dem til dine egne formål. Dog kan anmodning om sletning af dine personoplysninger også medføre tab af adgang til Game Launcher. Vi sletter ikke oplysninger, som vi er pålagt ved lov at opbevare.

For at fremsætte en anmodning vedrørende dine rettigheder eller en forespørgsel, bedes du kontakte os ved hjælp af de metoder, der er angivet i afsnittet KONTAKT OS.
