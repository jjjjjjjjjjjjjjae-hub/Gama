Personvernmerknad for Game Launcher


Ikrafttredelsesdato: 24. august 2021

Hos Samsung Electronics Co., Ltd. («Samsung»), vet vi hvor viktig personvern er for kundene våre. Samsung er databehandlingsansvarlig for Game Launcher. Vi laget denne personvernmerknaden for å passe på at du forstår hvordan vi innhenter og bruker dine personopplysninger.

Vår personvernerklæring på https://account.samsung.com/membership/terms/privacypolicy gjelder også for din bruk av Game Launcher. Vår personvernerklæring inneholder mer informasjon om hvordan vi bruker dataene dine. Den inkluderer også informasjon om rettighetene dine og hvordan du kontakter oss. Les personvernerklæringen i tillegg til denne personvernmerknaden. Denne personvernmerknaden skal imidlertid alltid ha forrang over personvernerklæringen i når det gjelder hvordan vi bruker informasjonen din for Game Launcher.




HVILKEN INFORMASJON INNHENTER VI?

Gjennom Game Launcher innhenter og beholder Samsung informasjon om deg på ulike måter.

Informasjon du gir til oss direkte

• Når du logger inn på din Samsung account, din Samsung account-informasjon, som din Samsung account-ID, profilbilde, alder og land. Og
• For å tilby svar når du kontakter kundeservice, ber vi om informasjon for å bekrefte identiteten din og produktet du kontakter oss om. Din kommunikasjon med kundeservicemedarbeiderne eller andre vil bli overført gjennom våre systemer.

Informasjon om din bruk av Game Launcher

I tillegg til informasjonen du oppgir, vil vi innhente informasjon om din bruk av vår Game Launcher gjennom programvare på enhetene dine og på andre måter. Vi vil innhente:

• Informasjon om dine Game Launcher-enheter, unike enhetsidentifikatorer, som IMEI, MCC (landskode for mobil), MNC (mobilnettverkskode), operativsystemversjoner og innstillinger for Game Launcher og enhetene du bruker for å få tilgang til Game Launcher.
• Informasjon om diagnoser, teknikk, feil og bruk, inkludert informasjon om hvordan, når og hvor lenge du bruker Game Launcher, sammen med annen informasjon om din bruk av Game Launcher, som hvordan du samhandler med innholdet som tilbys gjennom Game Launcher. Vi innhenter også informasjon om spill vi anbefaler for deg og de spillene du spiller på eller gjennom Game Launcher, som titlene på installerte apper, pakkenavnet til spillet du spiller, og spill som er installert og slettet. Og
• Informasjon fra partnere: Når du spiller et bestemt spill, mottar vi informasjon om spillet du har åpnet eller begynt å spille (inkludert versjonen av spillet), hvor mange ganger du åpner eller begynner å spille spillet, og hvor lenge du spiller et bestemt spill. I tillegg innhenter vi tilpassede hendelser i spillene du spiller, som når du fullfører en opplæring, en score eller prestasjon, når du fullfører et nivå, og når du taper et nivå.

Vi kan også innhente annen informasjon om deg, enhetene og appene dine og din bruk av Game Launcher på måter som vi beskriver for deg på innhentingstidspunktet, eller på andre måter som du samtykker til.

Vi bruker tredjeparts analysetjenester som Firebase Analytics. Informasjonen vi innhenter, kan bli videreformidlet til eller bli innhentet direkte av disse leverandørene eller andre relevante tredjeparter som bruker informasjonen til for eksempel å evaluere din bruk av Game Launcher og for å hjelpe med å administrere Game Launcher. Hvis du vil vite mer om Firebase Analytics, besøk https://www.google.com/policies/privacy/partners/.



HVORDAN BRUKER VI DIN INFORMASJON?
Vi bruker informasjonen vi innhenter til følgende formål:

• å identifisere og godkjenne deg, slik at du kan bruke Game Launcher.
• å levere Game Launcher til deg.
• å forbedre og tilpasse din opplevelse med Game Launcher, som å levere innhold og svar skreddersydd for deg og den måten du samhandler med Game Launcher på (for eksempel ved å lagre dine personlige prestasjoner og fremgang) og sørge for innholdsanbefalinger.
• å koble din Samsung account til tilkoblede Samsung- eller tredjepartsenheter, apper, funksjoner eller tjenester.
• å svare på dine spørsmål, forespørsler og instruksjoner sendt via Game Launcher.
• å levere annonsering, inkludert tilpassede annonser, sponset innhold og salgsfremmende kommunikasjon (som via push-varsler).
• å gjøre det mulig for deg å samhandle med og bruke deltakende apper og tjenester fra tredjeparter.
• å drive, evaluere og forbedre Game Launcher og virksomheten vår (inkludert å utvikle nye produkter og tjenester, utvide og forbedre produktene og tjenestene våre, håndtere kommunikasjon, analysere våre produkter og tjenester og kundebasen vår, gjennomføre markedsundersøkelser, utføre dataanalyser samt føre og revidere regnskap og utføre andre interne funksjoner).
• å beskytte mot, identifisere og hindre svindel, andre kriminelle handlinger, krav og annet erstatningsansvar. Og
• å etterleve og håndheve gjeldende rettslige krav, relevante bransjestandarder og våre egne retningslinjer, inkludert denne personvernmerknaden og personvernerklæringen.

Game Launcher kan tilby muligheten til å koble til eller vise andre nettsteder, apper, tjenester og andre funksjoner, som å vise innhold du liker eller bruker fra Samsung eller tredjeparters tjenester. Tredjeparts nettsteder, apper, tjenester og funksjoner kan drive uavhengig av oss og kan ha sine egne personvernmerknader eller retningslinjer for personvern. Gå gjennom dem for å sikre at du forstår hvordan de behandler informasjonen din. I den utstrekning andre tilknyttede eller viste nettsteder, apper, tjenester eller andre funksjoner ikke eies eller kontrolleres av oss, er vi ikke ansvarlig for deres innhold, bruk eller personvernpraksis.

KONTOSAMMENKOBLING

Noen tredjepartstjenester kan tilby kontosammenkobling, som lar deg koble din Samsung account sammen med kontoen til tredjepartstjenester. Når kontoene dine er sammenkoblet, kan du muligens bruke noen av funksjonene fra tredjepartstjenesten uten å installere tredjepartens app. Hvis du for eksempel logger inn på din Samsung account og fullfører sammenkoblingsprosedyren i den aktuelle tjenesten, vil du muligens kunne bruke deler av tredjepartens funksjoner tilbudt av en tredjepart.

Hver gang en konto sammenkobles, vil visse personlige data sendes til den tredjeparten, og den tredjeparten vil sende visse personlige data til Samsung. Før du kobler sammen kontoer, vil vi informere deg spesifikt om hvilke personlige data som vil bli delt, og for hvilket formål de spesifikke personlige dataene vil bli delt.

Når du kobler sammen kontoer, vil vi også informere deg om hvordan du kobler kontoer fra hverandre.

Samsung behandler personopplysninger til formålene som er beskrevet ovenfor. Samsungs rettslige grunnlag for å behandle personopplysninger skisseres nedenfor:

 Holde vårt løfte til deg

GDPR Artikkel 6(1)(b)

   • Identifisere og autentisere deg

   • Levere Game Launcher til deg, som for å la deg spille ulike spill på eller gjennom Game Launcher

   • La deg samhandle med annet innhold fra Samsung og tredjeparter

 

For å fremme våre virksomhetsinteresser

GDPR Artikkel 6(1)(f)

   • Drive, evaluere og forbedre Game Launcher og vår virksomhet

   • Levere deg tilpasset innhold, som å lagre prestasjoner og fremgang

   • Levere innholdsanbefalinger til deg

   • Svare på dine forespørsler, instruksjoner og henvendelser

   • Opprettholde adekvate sikkerhetstiltak

   • Beskytte mot erstatningsansvar, herunder å overholde bransjestandarder og håndheve våre retningslinjer

For å overholde loven og juridiske prosesser

GDPR Artikkel 6(1)(c)

   • Overholde gjeldende lovbestemmelser, forskrifter og rettskjennelser

 Med ditt samtykke

GDPR Artikkel 6(1)(a)

   • Levere markedsføringsmeldinger (som via push-meldinger)

   • La deg koble deg til din Samsung account med tredjepartssamtykke

   • Gi deg interessebaserte annonser

 





HVEM DELER VI INFORMASJONEN DIN MED?

Vi vil dele informasjonen din internt i virksomheten vår og med følgende enheter, men kun for de formålene som beskrives ovenfor.

• Forretningspartnere: partnere som vi arbeider sammen med for å levere Game Launcher, som tredjepartstjenester som gjør det mulig for deg å koble sammen deres konto med din Samsung account. Disse forretningspartnerne kontrollerer og administrerer dine personopplysninger.
• Tjenesteleverandører: nøye utvalgte selskaper som leverer tjenester til eller på vegne av oss, som for eksempel nettsky-plattformer som opprettholder data på våre vegne. Disse leverandørene er også forpliktet til å beskytte dine opplysninger.
• Andre parter når loven krever det eller når det er nødvendig for å beskytte Game Launcher. For eksempel kan det være nødvendig i henhold til lovbestemmelse, rettslig prosess eller rettskjennelse fra myndighetene å offentliggjøre opplysningene dine. Myndighetene vil også kunne søke å få informasjonen din fra oss for formål knyttet til rettshåndhevelse, rikets sikkerhet, terrorbekjempelse, eller andre spørsmål som er relatert til allmennhetens sikkerhet.
• Andre parter i forbindelse med selskapstransaksjoner: Vi vil kunne avsløre dine opplysninger til en tredjepart som del av en fusjon eller overføring, oppkjøp eller salg, eller i tilfelle av konkurs. Og
• Andre parter med ditt samtykke eller etter din anvisning: i tillegg til de offentliggjøringer som beskrives i denne Personvernmerknaden, vil vi kunne dele informasjon om deg med tredjeparter når du separat gir samtykke til eller ber om slik deling.



HVOR LENGE BEHOLDER VI INFORMASJONEN DIN OG HVOR GÅR DEN?

Hvor lenge dine data vil bli beholdt avhenger av hva som er det rettslige grunnlaget for å behandle dataene dine. For eksempel vil data vi behandler for å gjennomføre vår kontrakt med deg kreve at vi beholder dataene gjennom hele den tiden du bruker Game Launcher. Så lenge du er en aktiv bruker av Game Launcher, vil vi beholde og behandle disse dataene. Data som dine kontoopplysninger og enhetsinformasjon og identifikatorer faller inn under denne kategorien.

Data vi behandler for å fremme våre forretningsinteresser, som din bruk av Game Launcher, beholdes bare så lenge de trengs for de formålene de ble innsamlet for. For eksempel kan disse dataene være innsamlet for å utføre analyser slik at vi kan utvikle forbedringer i Game Launcher, eller vi vil kunne behandle data for å beholde registreringer over forespørsler du har kommet med eller angående Game Launcher for å forbedre din kundeserviceopplevelse.

I tillegg vil vi også kunne behandle dine data på grunnlag av ditt samtykke. Hvis du trekker tilbake ditt samtykke, vil vi fortsette å behandle data som er innsamlet før din tilbaketrekking inntil de formålene dataene ble innsamlet for er oppnådd.

Vennligst merk deg at selv om vi tar sikte på å beholde dine data i en slik periode som beskrives ovenfor, kan det være at dine data behandles lenger enn dette i henhold til gjeldende lov. For eksempel, hvis en spesifikk lovbestemmelse gir hjemmel for at vi må kreve visse opplysninger, vil vi rette oss etter dette og beholde disse opplysningene frem til den aktuelle oppbevaringsperioden utløper.

Så lenge dine data beholdes av oss, vil dine data alltid være underlagt formålstjenlige sikkerhetstiltak.

Din bruk av Game Launcher vil innebære overføring, lagring og behandling av dine personopplysninger i andre land. Dette inkluderer, uten begrensning, EØS-land, Sør-Korea og USA. Alle internasjonale dataoverføringer er underlagt juridiske krav for å sikre at dine personopplysninger blir behandlet trygt og slik som du vil forvente. Slike tiltak omfatter bruken av standard kontraktklausuler for å beskytte overføringen av data utenfor EØS. For å be om mer informasjon eller få en kopi av kontraktsavtalene som er på plass, se KONTAKT OSS-delen i denne personvernmerknaden.




HVOR LENGE HOLDER VI INFORMASJONEN DIN SIKKER?

Vi tar personvern svært alvorlig. Vi har satt på plass fysiske og tekniske sikkerhetstiltak for å holde informasjonen vi innhenter sikker. Vær imidlertid oppmerksom på at selv om vi tar rimelige forholdsregler for å beskytte informasjonen din, er ingen nettsteder, Internett-overføringer, datasystemer eller trådløse tilkoblinger fullstendig sikre.




HVA ER DINE RETTIGHETER?

Dine personopplysninger tilhører deg. Du kan be oss om å oppgi detaljer om hva vi har innhentet og du kan be oss om å slette det eller korrigere unøyaktigheter. Du kan også be oss om å begrense eller innskrenke behandlingen, delingen eller overføringen av dine personopplysninger, samt å gi deg de personopplysningene som vi har innhentet, slik at du kan bruke dem til dine egne formål. Imidlertid kan det å be om sletting av dine personopplysninger også føre til tap av tilgang til Game Launcher. Vi sletter ikke data som vi er pålagt å beholde etter lov.

For å gjøre en forespørsel om dine rettigheter eller for å utføre en henvendelse, kan du ta kontakt gjennom metodene beskrevet i KONTAKT OSS-delen.
