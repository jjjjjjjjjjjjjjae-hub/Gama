Game Launcher – Integritetsmeddelande


Giltighetsdatum: 24 augusti 2021

Samsung Electronics Co., Ltd. (”Samsung”) vet hur viktig integriteten är för våra kunder. Samsung är personuppgiftsansvarig för Game Launcher. Vi har upprättat detta integritetsmeddelande för att säkerställa att du förstår hur vi samlar in och använder dina personuppgifter.

Vår sekretesspolicy, som finns på https://account.samsung.com/membership/terms/privacypolicy, gäller också för din användning av Game Launcher. I vår sekretesspolicy finns mer information om hur vi använder dina uppgifter. Den innehåller dessutom information om dina rättigheter och hur du kan kontakta oss. Läs igenom vår Sekretesspolicy utöver detta integritetsmeddelande. Detta integritetsmeddelande ska dock alltid gälla över sekretesspolicyn i relation till hur vi använder dina uppgifter i Game Launcher.




VILKA UPPGIFTER SAMLAR VI IN?

Genom Game Launcher, inhämtar och lagrar Samsung information om dig på olika sätt.

Uppgifter som du tillhandahåller direkt

• När du loggar in på ditt Samsung account och information på ditt Samsung account, såsom ditt Samsung account-ID, smeknamn, profilbild, ålder och land; och
• För att ge dig svar när du kontaktar kundtjänst ber vi om uppgifter för att bekräfta din identitet och den produkt du kontaktar oss om. När du kommunicerar med vår kundtjänst eller andra överförs kommunikationen via våra system.

Information om din användning av Game Launcher

Utöver de uppgifter du själv tillhandahåller samlar vi även in uppgifter om din användning av Game Launcher genom programvara på dina enheter och på andra sätt. Vi samlar in:

• Information om dina Game Launcher-aktiverade enheter, unika enhetsidentifierare såsom krypterade IMEI-nummer, mobil landskod (MCC), mobilnätskod (MNC), operativsystemversioner och inställningar för Game Launcher och de enheter som du använder för att få åtkomst till Game Launcher;
• Diagnostisk, teknisk, fel- och användningsinformation, däribland om hur, när och hur länge du använder Game Launcher, tillsammans med annan information om din användning av Game Launcher, till exempel hur du interagerar med innehåll som erbjuds via Game Launcher. Vi samlar också in information om spel som vi rekommenderar till dig och om de spel som du spelar på eller via Game Launcher, t.ex. titlar på installerade appar, paketnamnet på det spel du spelar samt installerade och raderade spel; och
• Information från partners: När du spelar ett visst spel får vi information om spelet du har öppnat eller börjat spela (inklusive spelets version), hur många gånger du öppnar eller börjar spela spelet och hur länge du spelar ett visst spel. Vi samlar dessutom in anpassade händelser i de spel du spelar, t.ex. när du slutför en handledning, poäng eller prestation och när du klarar eller misslyckas på en nivå.

Vi kan även samla in annan information om dig, dina enheter och appar samt din användning av Game Launcher genom metoder som beskrivs när informationen samlas in eller erhålls på andra sätt med ditt samtycke.

Vi kan använda analystjänster från tredje parter som till exempel Firebase Analytics. De uppgifter vi erhåller kan lämnas ut till eller samlas in direkt av dessa leverantörer och andra relevanta tredje parter som använder uppgifterna, till exempel för att utvärdera användningen av Game Launcher och för att hjälpa till att administrera Game Launcher. För mer information om Firebase Analytics kan du besöka https://www.google.com/policies/privacy/partners/.



HUR ANVÄNDER VI DINA UPPGIFTER?
Vi använder de uppgifter vi samlar in för följande ändamål:

• för att identifiera och verifiera dig så att du kan använda Game Launcher;
• för att göra Game Launcher tillgänglig för dig;
• för att förbättra och anpassa din upplevelse med Game Launcher, såsom att leverera innehåll och svar som är skräddarsydda för dig och ditt sätt att interagera med Game Launcher (t.ex. genom att spara dina personliga prestationer och framsteg) och tillhandahålla innehållsrekommendationer;
• för att länka ditt Samsung account till anslutna Samsung- eller tredjepartsenheter, appar, funktioner eller tjänster;
• för att besvara dina förfrågningar och ge dig instruktioner genom Game Launcher;
• för att leverera reklam, inklusive anpassade annonser, sponsrat innehåll och kampanjkommunikation (t.ex. via pushnotiser);
• för att låta dig interagera med och använda appar och tjänster från tredje part;
• för att driva, utvärdera och förbättra Game Launcher och vår verksamhet (däribland utveckla nya produkter och tjänster, förbättra våra produkter och tjänster, hantera vår kommunikation, analysera våra produkter, tjänster och vår kundbas, genomföra marknadsundersökningar, utföra dataanalyser samt utföra redovisning, revision och andra interna funktioner);
• för att skydda mot, identifiera och förhindra bedrägeri och annan kriminell aktivitet, anspråk och annat ansvar; och
• för att efterleva och verkställa tillämpliga rättsliga krav, relevanta branschstandarder och våra policyer, inklusive detta integritetsmeddelande och sekretesspolicyn.

Game Launcher kan komma att tillhandahålla möjligheten att ansluta till eller visa andra webbplatser, appar, tjänster eller andra funktioner såsom att visa innehåll du gillar eller använder från Samsung eller tredjepartstjänster. Tredje parts webbplatser, appar, tjänster och funktioner kan drivas fristående från oss och kan ha sina egna integritetsmeddelanden eller policyer. Granska dem för att förstå hur de behandlar dina uppgifter. Vi ansvarar inte för innehållsanvändning eller integritetspraxis för andra eventuella länkade eller visade webbplatser, appar, tjänster eller andra funktioner som inte ägs eller kontrolleras av oss.

KONTOLÄNKNING

Vissa tredjepartstjänster kan komma att erbjuda kontolänkning, som låter dig ansluta ditt Samsung account med tredjepartstjänsters konton. När ditt konto väl har länkats kan du ha möjlighet att använda vissa av de funktioner som tillhandahålls av tredjepartstjänsten utan installation av applikation från tredje part. Om du exempelvis loggar in på ditt Samsung account och avslutar länkningsförfarandet i den tillämpliga tjänsten, kan du använda delar av de funktioner som tillhandahålls av en tredje part.

Varje gång ett konto länkas kommer vissa personuppgifter att skickas till den tredje parten och den tredje parten kommer att skicka vissa personuppgifter till Samsung. Innan kontolänkning sker kommer vi specifikt att informera dig om vilka personuppgifter som kommer att delas och för vilket ändamål de särskilda personuppgifterna kommer att delas.

När du länkar konton informerar vi dig också om hur du kan avlänka konton.

Samsung bearbetar personuppgifter för de ändamål som beskrivs ovan. Samsungs rättsliga grund för att behandla personuppgifter beskrivs nedan:

För att främja våra affärsintressen

GDPR Artikel 6(1)(f)

   • För att ge dig skräddarsytt innehåll, t.ex. genom att spara dina personliga prestationer och framsteg

   • För att ge dig rekommendationer om innehåll

   • Svara på dina begäranden, instruktioner och förfrågningar

   • Upprätthålla lämpliga säkerhetsåtgärder

   • Skydda mot ansvar, inklusive att följa branschstandarder och upprätthålla våra policyer

För att efterleva lagar och rättsliga processer

GDPR Artikel 6(1)(c)

   • Följa tillämpliga stadgar och förordningar och domstolsbeslut

 Med ditt samtycke

GDPR Artikel 6(1)(a)

   • Skicka marknadsföringsmeddelanden (t.ex. via pushnotiser)

   • Låta dig länka ditt Samsung account med tredjepartsinnehåll

MED VEM DELAR VI DINA UPPGIFTER?

Vi lämnar ut din information internt inom vårt företag samt till följande aktörer, men endast för de ändamål som beskrivs ovan.

• Affärspartners: partners som vi samarbetar med för att tillhandahålla Game Launcher till dig, t.ex. tredjepartstjänster som gör att du kan länka deras konto till ditt Samsung account. Dessa affärspartner kontrollerar och hanterar dina personuppgifter;
• Tjänsteleverantörer: noggrant utvalda företag som utför tjänster åt oss eller på uppdrag av oss, som till exempel molntjänstplattformar som upprätthåller data för vår räkning. Även dessa leverantörer har åtagit sig att skydda dina personuppgifter;
• Andra parter när det krävs enligt lag eller är nödvändigt för att skydda Game Launcher. Det kan till exempel bli nödvändigt enligt lag, juridiska processer eller domstolsorder från statliga myndigheter att lämna ut din information. De kan också söka få dina uppgifter från oss för brottsbekämpande ändamål, nationell säkerhet, terrorismbekämpning eller annat som rör den allmänna säkerheten;
• Andra parter i samband med bolagstransaktioner: vi kan lämna ut din information till en tredje part inom ramen för en sammanslagning eller överlåtelse, ett uppköp eller en försäljning, eller i händelse av konkurs; och
• Andra parter med ditt samtycke eller enligt anvisning från dig: utöver de avslöjanden som beskrivs i detta integritetsmeddelande kan vi dela information om dig med tredje parter när du separat samtycker till eller begär att vi ska göra detta.



HUR LÄNGE BEHÅLLER VI DIN INFORMATION OCH VART TAR DEN VÄGEN?

Hur länge dina uppgifter lagras beror på den rättsliga grund som åberopas för att behandla dina uppgifter. Uppgifter som vi behandlar för att utföra vårt kontrakt med dig kräver till exempel att vi lagrar uppgifterna under hela den tid du använder din Game Launcher. Vi lagrar och behandlar dessa uppgifter så länge du är en aktiv användare av Game Launcher. Uppgifter som din kontoinformation, enhetsinformation och identifierare ingår i denna kategori.

Uppgifter som vi behandlar för att marknadsföra våra affärsintressen, till exempel din användning av Game Launcher, lagras endast så länge som det behövs för de ändamål för vilka de samlades in. Dessa uppgifter kan till exempel samlas in för att utföra analyser så att vi kan utveckla förbättringar av Game Launcher, eller så kan vi behandla uppgifter för att föra register över förfrågningar du gjort genom eller om Game Launcher för att förbättra din upplevelse av vår kundtjänst.

Dessutom kan vi även behandla dina uppgifter baserat på ditt samtycke. Om du återkallar ditt samtycke, kommer vi att fortsätta behandla uppgifter som samlats in före ditt återkallande tills de ändamål för vilka sådana uppgifter samlades in har uppnåtts.

Observera att även om vi strävar efter att lagra dina uppgifter under den tidsperiod som beskrivs ovan, kan dina uppgifter bli behandlade under en längre tid, enligt gällande lag. Om till exempel en specifik stadga kräver att vi lagrar en viss uppgift, kommer vi att följa detta och lagra den uppgiften tills den föreskrivna lagringsperioden upphör.

Så länge dina uppgifter lagras av oss kommer dina uppgifter alltid att vara föremål för lämpliga skyddsåtgärder.

Din användning av Game Launcher inkluderar överföring, lagring och behandling av dina personuppgifter till eller i andra länder. Sådana länder inkluderar, utan begränsning, länder i det Europeiska ekonomiska samarbetsområdet, Sydkorea och USA. All överföring av uppgifter till andra länder regleras av lagstadgade krav för att säkerställa att dina personuppgifter behandlas säkert och enligt vad du skulle kunna förvänta dig. Sådana säkerhetsåtgärder innefattar användning av standardavtalsklausuler för att skydda överföringen av personuppgifter utanför EES. Om du vill ha mer information eller få en kopia av de avtal som gäller, se avsnittet KONTAKTA OSS i detta integritetsmeddelande.




HUR SKYDDAR VI DINA UPPGIFTER?

Vi tar dataskydd på stort allvar. Vi har infört fysiska och tekniska skyddsåtgärder för att hålla de personuppgifter som vi samlar in säkra. Observera dock att även om vi vidtar rimliga åtgärder för att skydda dina personuppgifter, är inga webbplatser, internetöverföringar, datorsystem eller trådlösa anslutningar helt säkra.




VILKA RÄTTIGHETER HAR DU?

Dina personuppgifter tillhör dig. Du kan be oss att tillhandahålla information om vilka personuppgifter vi har samlat in och du kan be oss att radera dem eller korrigera eventuella felaktigheter. Du kan också be oss att begränsa behandlingen, delningen eller överföringen av dina personuppgifter, samt att förse dig med dina personuppgifter som vi har samlat in, så att du kan använda dem för dina egna ändamål. Om du begär att vi raderar dina personuppgifter kan det emellertid leda till att du inte kan få åtkomst till Game Launcher. Vi kommer inte att radera uppgifter som vi enligt lag är skyldiga att behålla.

För att skicka en begäran avseende dina rättigheter eller för att göra en förfrågan kan du kontakta oss med hjälp av de metoder som beskrivs i avsnittet KONTAKTA OSS.
