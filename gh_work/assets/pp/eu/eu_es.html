Game Launcher - Pribatutasun oharra


Azken eguneratzea: 2020ko otsailaren 17

Samsung Electronics Co., Ltd. konpainian badakigu zer nolako garrantzia duen pribatutasunak gure bezeroentzat. Game Launcher-en datuen kontrola daukagu, eta Pribatutasun Ohar hau egin dugu ziurtatzeko ulertzen duzula nola bildu eta erabiltzen dugun zure informazio pertsonala.

Gure Pribatutasun Politika ere badagokio Game Launcher-en erabilerari (“Zerbitzua”). Gure Pribatutasun Politikak badakar informazio gehiago, nola erabiltzen ditugun zure datuak. Gurekin harremanetan jartzeko informazioa ere badakar. Mesedez, irakur ezazu gure Pribatutasun Politika, Pribatutasun Ohar honez gain. Edonola ere, Pribatutasun Ohar hau beti gailentzen zaio Pribatutasun Politikari.




ZER INFORMAZIO BILTZEN DUGU?

Game Launcher-en bidez, zuri buruzko informazioa jasotzen eta gordetzen dugu hainbat modutan.

Zuzenean ematen diguzun informazioa

• Samsung Account-ean saioa hasten duzunean, zure Samsung Account-eko IDa.
• Bezeroentzako zerbitzuekin harremanetan jartzen zarenean, erantzunak emateko, zure informazioa eskatuko dizugu, zure nortasuna eta harremanetan jartzearen produktua egiaztatzeko.
• Bezeroentzako zerbitzuko langileekin edo beste batzuekin harremanetan jartzen bazara, zure komunikazioak gure sistemaren bitartez helaraziko dira.

Game Launcher-en erabilerari buruzko informazioa

• Gailuko informazioa: Zure gailuaren modeloa, zenbaki identifikatzaile bakarrak, MCC zenbakia (Mugikorraren Herrialdeko Kodea), MNC zenbakia (Mugikorraren Sareko Kodea), sistema eragilearen bertsioak, eta Game Launcher-era sartzeko erabiltzen dituzun gailuen ezarpenak.
• Erregistroaren informazioa: Diagnostikoa, teknikoa, akatsak eta erabileraren informazioa, hala nola Game Launcher erabiltzen duzun denbora eta iraupena, instalatutako aplikazioaren titulua, jolasten ari zaren jokoaren pakete izena, zerbitzu erabiliaren grabazioa eta gertaeraren erregistroa, jokoaren instalazioa eta ezabapenaren inguruan.
• Game Launcher-en erabilera dela eta, beste informazioa, hala nola Game Launcher-en bidez eskaintzen den edukiarekin daukazun interakzioa.




ZERTARAKO ERABILTZEN DUGU ZURE INFORMAZIOA?

• Game Launcher erabili ahal izateko identifikazioa eta autentifikazioa.
• Eskatzen dituzun zerbitzuak eta eginbideak emateko.
• Aurreko jardueretan oinarrituta, mugikorretako jolasen gomendio pertsonalak emateko.
• Zure webguneetan, hirugarrenen webguneetan eta online plataformetan (esaterako, sare sozialetako guneak) interesa dakizkizukeen publizitate, sustapen eta eskaintza pertsonalizatuak eskaintzea.
• Sustapenak eta eskaintzak emateko merkataritza jakinarazpenen bidez, aparteko baimena eman diguzunean soilik.
• Zuk hirugarrenen aplikazioekin eta zerbitzuekin elkarreragiteko eta horiek erabiltzeko aukera izateko.
• Gure produktuak eta zerbitzuak aztertzeko; hartara, gure bezeroak hobeto ulertzen laguntzeko, komunikazio, zerbitzu eta esperientzia garrantzitsuenak eskaintze aldera.
• Gure produktuen eta zerbitzuen gainean duzun iritzia galdetzeko eta bezeroei inkestak egiteko, aparteko baimena eskatuta, behar denean.
• Game Launcher-en software eguneraketak, zerbitzu mantenuak eta laguntza emateko.
• Legea eta lege prozesuak betetzeko.
• Samsung Electronics-en edo bere filialen, bazkide komertzialen edo bezeroen eskubideak, ondasunak edo segurtasuna babesteko.




KONTUAK LOTZEA

Baliteke hirugarrenen zerbitzu batzuek kontuak lotzeko aukera eskaintzea. Aukera horren bidez, Samsung Account eta hirugarrenen zerbitzuetako kontua lotu ditzakezu. Kontuak lotutakoan, hirugarrenaren aplikazioa instalatu beharrik gabe erabili ahal izango dituzu hirugarrenen zerbitzuko eginbide batzuk. Adibidez, Samsung Account-en saioa hasten baduzu eta Zerbitzuan amaitzen baduzu kontuak lotzeko prozedura, hirugarrenek eskainitako Discord Service zerbitzuko funtzio batzuk erabili ahal izango dituzu.

Kontuak lotzen diren bakoitzean, Samsung-ek zenbait datu pertsonal bidaliko dizkio hirugarren horri, eta hirugarren horrek zenbait datu pertsonal bidaliko dizkio Samsung-i. Kontuak lotu aurretik, berariaz jakinaraziko dizugu zer datu pertsonal partekatuko ditugun, baita datu pertsonal horiek zer helburutarako partekatuko ditugun ere.

Game Launcher-eko Discord-en Ezarpenak menuak ken diezaiekezu lotura kontuei.

Samsung-ek arestian adierazi diren helburuetarako tratatzen du informazio pertsonala. Samsung-ek informazio pertsonala prozesatzeko baliatzen dituen lege oinarrien arabera, helburu hauetarako prozesatzen da:
(i) Hitz emandakoa bete ahal izateko, adibidez, Samsung Daily erabili ahal izan dezazun;
(ii) Gure negozio interesak sustatzeko (esate baterako, zurekin dugun harremana kudeatzeko eta bezeroentzako zerbitzuen esperientzia hobetzeko), gure webguneko, hirugarrenen webguneetako eta online plataformetako jakinarazpenak barne; eta
(iii) Legea eta lege prozesuak betetzeko; eta
(iv) Zure baimenarekin egingo dugu (esate baterako, merkataritza jakinarazpen zuzenak bidaltzeko). Baimena ematen baduzu, beti egin dezakezu atzera, gurekin harremanetan jarriz gure Pribatutasun Politikaren JAR ZAITEZ HARREMANETAN atalean.

Game Launcher-a erabiltzen duzunean, zure datuak eraman, gorde eta prozesatuko dira Europako esparru ekonomikoko herrialdeetan, AEBetan, Koreako Errepublikan edo beste herrialdetan.




PRIBATUTASUN OHAR HONEN EGUNERAKETAK

Pribatutasun Ohar hau eguneratu ahal da zure Game Launcher-ean zure informazioa nola gorde eta prozesatzeko izan ditugun aldaketen berri emateko, edo lege aplikagarrietan eman diren aldaketen berri emateko Pribatutasun Ohar honen hasieran erakusten da agiria azken aldiz eguneratu den data.
